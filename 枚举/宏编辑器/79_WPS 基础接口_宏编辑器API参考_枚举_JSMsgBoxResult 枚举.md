# JSMsgBoxResult 枚举

用户点击MsgBox弹出消息框的按钮。

| 名称     | 值  | 说明 |
|----------|-----|------|
| jsOK     | 1   | 确定 |
| jsCancel | 2   | 取消 |
| jsAbort  | 3   | 中止 |
| jsRetry  | 4   | 重试 |
| jsIgnore | 5   | 忽略 |
| jsYes    | 6   | 是   |
| jsNo     | 7   | 否   |
