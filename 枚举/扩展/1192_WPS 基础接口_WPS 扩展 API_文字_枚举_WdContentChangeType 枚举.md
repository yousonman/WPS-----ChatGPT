# WdContentChangeType 枚举

指定文档变化的类型。

| 名称              | 值  | 说明               |
|-------------------|-----|--------------------|
| wdContentInsert   | 0   | 文本内容插入。     |
| wdContentDelete   | 1   | 文本内容删除。     |
| wdContentStyle    | 2   | 文本应用样式变化。 |
| wdContentHeading  | 3   | 文本应用标题变化。 |
| wdContentParaProp | 4   | 段落属性变化。     |
