# WdDocBarcodeMode 枚举

指定公文二维码宽高模式。

| 名称          | 值  | 说明                         |
|---------------|-----|------------------------------|
| WdAutoSize    | 0   | 由系统自动指定二维码的大小。 |
| WdFixedWidth  | 1   | 固定二维码的宽度。           |
| WdFixedHeight | 2   | 固定二维码的高度。           |
