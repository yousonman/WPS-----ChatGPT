# XlDirection 枚举

指定移动的方向。

| 名称      | 值    | 说明   |
|-----------|-------|--------|
| xlDown    | -4121 | 向下。 |
| xlToLeft  | -4159 | 向左。 |
| xlToRight | -4161 | 向右。 |
| xlUp      | -4162 | 向上。 |
