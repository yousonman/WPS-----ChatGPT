# XlDisplayBlanksAs 枚举

指定在图表上绘制空单元格的方式。

| 名称           | 值  | 说明               |
|----------------|-----|--------------------|
| xlInterpolated | 3   | 将值内插入图表。   |
| xlNotPlotted   | 1   | 不绘制空单元格。   |
| xlZero         | 2   | 空单元格绘制为零。 |
