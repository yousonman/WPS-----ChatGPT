# XlDisplayDrawingObjects 枚举

指定形状的显示方式。

| 名称            | 值    | 说明           |
|-----------------|-------|----------------|
| xlDisplayShapes | -4104 | 显示所有形状。 |
| xlHide          | 3     | 隐藏所有形状。 |
| xlPlaceholders  | 2     | 仅显示占位符。 |
