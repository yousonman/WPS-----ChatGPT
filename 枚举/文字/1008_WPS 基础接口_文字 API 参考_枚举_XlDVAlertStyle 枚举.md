# XlDVAlertStyle 枚举

指定验证过程中显示的消息框所用的图标。

| 名称                    | 值  | 说明       |
|-------------------------|-----|------------|
| xlValidAlertInformation | 3   | 信息图标。 |
| xlValidAlertStop        | 1   | 停止图标。 |
| xlValidAlertWarning     | 2   | 警告图标。 |
