# XlEditionFormat 枚举

指定发布版本的格式。此枚举仅用于 Macintosh，因此不应使用。

| 名称   | 值  | 说明                      |
|--------|-----|---------------------------|
| xlBIFF | 2   | 二进制交换文件格式。      |
| xlPICT | 1   | 图元文件图片结构 (.wmf)。 |
| xlRTF  | 4   | RTF 格式 (.rtf)。         |
| xlVALU | 8   | VALU。                    |
