# XlEditionOptionsOption 枚举

此枚举仅用于 Macintosh，因此不应使用。

| 名称               | 值  | 说明                         |
|--------------------|-----|------------------------------|
| xlAutomaticUpdate  | 4   | 自动更新。                   |
| xlCancel           | 1   | 取消。                       |
| xlChangeAttributes | 6   | 更改属性。                   |
| xlManualUpdate     | 5   | 手动更新。                   |
| xlOpenSource       | 3   | 打开源。                     |
| xlSelect           | 3   | 选择。                       |
| xlSendPublisher    | 2   | 发送到 Microsoft Publisher。 |
| xlUpdateSubscriber | 2   | 更新订阅服务器。             |
