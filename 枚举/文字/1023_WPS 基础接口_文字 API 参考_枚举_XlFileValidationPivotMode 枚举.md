# XlFileValidationPivotMode 枚举

指定如何验证数据透视表的数据缓存。

| 名称                         | 值  | 说明                                                       |
|------------------------------|-----|------------------------------------------------------------|
| xlFileValidationPivotDefault | 0   | 验证 PivotOptions 注册表设置指定的数据缓存的内容（默认）。 |
| xlFileValidationPivotRun     | 1   | 验证所有数据缓存的内容，而不考虑注册表设置。               |
| xlFileValidationPivotSkip    | 2   | 不验证数据缓存的内容。                                     |
