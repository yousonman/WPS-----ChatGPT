# XlFillWith 枚举

指定如何复制区域。

| 名称               | 值    | 说明             |
|--------------------|-------|------------------|
| xlFillWithAll      | -4104 | 复制内容和格式。 |
| xlFillWithContents | 2     | 仅复制内容。     |
| xlFillWithFormats  | -4122 | 仅复制格式。     |
