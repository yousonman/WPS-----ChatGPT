# XlFixedFormatQuality 枚举

指定以不同固定格式保存的电子表格的质量。

| 名称              | 值  | 说明     |
|-------------------|-----|----------|
| xlQualityMinimum  | 1   | 最低质量 |
| xlQualityStandard | 0   | 标准质量 |
