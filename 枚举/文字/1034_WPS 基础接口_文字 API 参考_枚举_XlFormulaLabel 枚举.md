# XlFormulaLabel 枚举

为指定区域指定公式标签类型。

| 名称           | 值    | 说明             |
|----------------|-------|------------------|
| xlColumnLabels | 2     | 仅列标签。       |
| xlMixedLabels  | 3     | 行标签和列标签。 |
| xlNoLabels     | -4142 | 无标签。         |
| xlRowLabels    | 1     | 仅行标签。       |
