# XlHAlign 枚举

指定对象的水平对齐方式。

| 名称                          | 值    | 说明             |
|-------------------------------|-------|------------------|
| xlHAlignCenter                | -4108 | 居中。           |
| xlHAlignCenterAcrossSelection | 7     | 跨列居中。       |
| xlHAlignDistributed           | -4117 | 分散对齐。       |
| xlHAlignFill                  | 5     | 填充。           |
| xlHAlignGeneral               | 1     | 按数据类型对齐。 |
| xlHAlignJustify               | -4130 | 两端对齐。       |
| xlHAlignLeft                  | -4131 | 靠左。           |
| xlHAlignRight                 | -4152 | 靠右。           |
