# XlHighlightChangesTime 枚举

指定共享工作簿中显示的一组更改。

| 名称              | 值  | 说明                                   |
|-------------------|-----|----------------------------------------|
| xlAllChanges      | 2   | 显示所有更改。                         |
| xlNotYetReviewed  | 3   | 仅显示还未审阅的更改。                 |
| xlSinceMyLastSave | 1   | 显示上次保存后最后一个用户进行的更改。 |
