# XlInsertShiftDirection 枚举

指定插入时单元格的移动方向。

| 名称           | 值    | 说明             |
|----------------|-------|------------------|
| xlShiftDown    | -4121 | 向下移动单元格。 |
| xlShiftToRight | -4161 | 向右移动单元格。 |
