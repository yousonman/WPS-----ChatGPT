# XlLineStyle 枚举

指定边框的线条样式。

| 名称            | 值    | 说明             |
|-----------------|-------|------------------|
| xlContinuous    | 1     | 实线。           |
| xlDash          | -4115 | 虚线。           |
| xlDashDot       | 4     | 点划相间线。     |
| xlDashDotDot    | 5     | 划线后跟两个点。 |
| xlDot           | -4118 | 点线。           |
| xlDouble        | -4119 | 双线。           |
| xlLineStyleNone | -4142 | 无线。           |
| xlSlantDashDot  | 13    | 倾斜的划线。     |
