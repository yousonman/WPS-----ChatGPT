# XlLinkInfoType 枚举

指定链接的类型。

| 名称                  | 值  | 说明              |
|-----------------------|-----|-------------------|
| xlLinkInfoOLELinks    | 2   | OLE 或 DDE 服务器 |
| xlLinkInfoPublishers  | 5   | 发布服务器        |
| xlLinkInfoSubscribers | 6   | 订阅服务器        |
