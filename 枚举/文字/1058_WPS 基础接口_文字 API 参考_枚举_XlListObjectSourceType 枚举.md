# XlListObjectSourceType 枚举

指定列表的当前源。

| 名称          | 值  | 说明                                                 |
|---------------|-----|------------------------------------------------------|
| xlSrcExternal | 0   | 外部数据源（Microsoft SharePoint Foundation 网站）。 |
| xlSrcQuery    | 3   | 查询                                                 |
| xlSrcRange    | 1   | 区域                                                 |
| xlSrcXml      | 2   | XML                                                  |
