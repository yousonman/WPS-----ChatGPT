# XlMailSystem 枚举

指定安装在主机上的邮件系统。

| 名称           | 值  | 说明               |
|----------------|-----|--------------------|
| xlMAPI         | 1   | 符合 MAPI 的系统   |
| xlNoMailSystem | 0   | 无邮件系统         |
| xlPowerTalk    | 2   | PowerTalk 邮件系统 |
