# XlMousePointer 枚举

指定 ET 中鼠标指针的外观。

| 名称             | 值    | 说明             |
|------------------|-------|------------------|
| xlDefault        | -4143 | 默认指针。       |
| xlIBeam          | 3     | I 形指针。       |
| xlNorthwestArrow | 1     | 西北向箭头指针。 |
| xlWait           | 2     | 沙漏型指针。     |
