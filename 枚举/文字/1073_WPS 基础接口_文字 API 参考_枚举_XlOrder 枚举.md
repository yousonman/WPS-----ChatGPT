# XlOrder 枚举

指定单元格的处理次序。

| 名称           | 值  | 说明                                       |
|----------------|-----|--------------------------------------------|
| xlDownThenOver | 1   | 向下处理行，然后向右逐个处理页或页面字段。 |
| xlOverThenDown | 2   | 向右逐个处理页或页面字段，然后向下处理行。 |
