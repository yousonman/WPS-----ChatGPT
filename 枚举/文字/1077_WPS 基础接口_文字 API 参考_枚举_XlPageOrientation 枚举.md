# XlPageOrientation 枚举

指定打印工作表时的页面方向。

| 名称        | 值  | 说明       |
|-------------|-----|------------|
| xlLandscape | 2   | 横向模式。 |
| xlPortrait  | 1   | 纵向模式。 |
