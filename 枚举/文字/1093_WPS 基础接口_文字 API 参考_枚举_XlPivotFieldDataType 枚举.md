# XlPivotFieldDataType 枚举

指定 **PivotTable** 字段中数据的类型。

| 名称     | 值    | 说明           |
|----------|-------|----------------|
| xlDate   | 2     | 包含一个日期。 |
| xlNumber | -4145 | 包含一个数字。 |
| xlText   | -4158 | 包含文本。     |
