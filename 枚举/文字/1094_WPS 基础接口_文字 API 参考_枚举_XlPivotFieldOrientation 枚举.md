# XlPivotFieldOrientation 枚举

指定字段在数据透视表中的位置。

| 名称          | 值  | 说明       |
|---------------|-----|------------|
| xlColumnField | 2   | 列字段。   |
| xlDataField   | 4   | 数据字段。 |
| xlHidden      | 0   | 隐藏字段。 |
| xlPageField   | 3   | 页字段。   |
| xlRowField    | 1   | 行字段。   |
