# XlPivotLineType 枚举

指定 PivotLine 的类型。

| 名称                  | 值  | 说明                           |
|-----------------------|-----|--------------------------------|
| xlPivotLineBlank      | 3   | 每组后的空行。                 |
| xlPivotLineGrandTotal | 2   | 总计行。                       |
| xlPivotLineRegular    | 0   | 带有透视项目的常规 PivotLine。 |
| xlPivotLineSubtotal   | 1   | 分类汇总行。                   |
