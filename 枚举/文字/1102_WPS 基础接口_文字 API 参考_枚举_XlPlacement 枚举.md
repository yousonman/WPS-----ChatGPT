# XlPlacement 枚举

指定对象附加到其下层单元格的方式。

| 名称           | 值  | 说明                         |
|----------------|-----|------------------------------|
| xlFreeFloating | 3   | 对象自由浮动。               |
| xlMove         | 2   | 对象随单元格移动。           |
| xlMoveAndSize  | 1   | 对象随单元格移动和调整大小。 |
