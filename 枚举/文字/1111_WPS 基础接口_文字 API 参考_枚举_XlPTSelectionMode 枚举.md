# XlPTSelectionMode 枚举

指定在结构化选择过程中可以在数据透视表中选择的内容。这些常数可以进行组合以选择多个类型。

| 名称           | 值  | 说明       |
|----------------|-----|------------|
| xlBlanks       | 4   | 空值       |
| xlButton       | 15  | 按钮       |
| xlDataAndLabel | 0   | 数据和标签 |
| xlDataOnly     | 2   | 数据       |
| xlFirstRow     | 256 | 第一行     |
| xlLabelOnly    | 1   | 标签       |
| xlOrigin       | 3   | 原点       |
