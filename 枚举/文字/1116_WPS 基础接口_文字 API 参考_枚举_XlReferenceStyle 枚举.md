# XlReferenceStyle 枚举

指定引用样式。

| 名称   | 值    | 说明                                   |
|--------|-------|----------------------------------------|
| xlA1   | 1     | 默认值。使用 xlA1 返回 A1 样式的引用。 |
| xlR1C1 | -4150 | 使用 xlR1C1 返回 R1C1 样式的引用。     |
