# XlRoutingSlipStatus 枚举

指定传送名单的状态。

| 名称                | 值  | 说明               |
|---------------------|-----|--------------------|
| xlNotYetRouted      | 0   | 还未发送传送名单。 |
| xlRoutingComplete   | 2   | 完成传送。         |
| xlRoutingInProgress | 1   | 正在传送。         |
