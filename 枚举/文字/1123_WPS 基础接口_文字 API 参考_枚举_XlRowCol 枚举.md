# XlRowCol 枚举

指定与特定数据系列对应的值是在行中还是在列中。

| 名称      | 值  | 说明               |
|-----------|-----|--------------------|
| xlColumns | 2   | 数据系列位于行中。 |
| xlRows    | 1   | 数据系列位于列中。 |
