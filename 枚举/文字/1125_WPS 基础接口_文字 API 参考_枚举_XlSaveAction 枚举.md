# XlSaveAction 枚举

如果将保存文件，则在文件关闭过程中进行指定。

| 名称               | 值  | 说明         |
|--------------------|-----|--------------|
| xlDoNotSaveChanges | 2   | 不保存更改。 |
| xlSaveChanges      | 1   | 保存更改。   |
