# XlSortMethod 枚举

指定排序类型。

| 名称     | 值  | 说明                                   |
|----------|-----|----------------------------------------|
| xlPinYin | 1   | 按字符的汉语拼音顺序排序。这是默认值。 |
| xlStroke | 2   | 按每个字符的笔划数排序。               |
