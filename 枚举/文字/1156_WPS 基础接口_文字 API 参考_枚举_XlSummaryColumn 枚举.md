# XlSummaryColumn 枚举

指定汇总列在大纲中的位置。

| 名称             | 值    | 说明                                 |
|------------------|-------|--------------------------------------|
| xlSummaryOnLeft  | -4131 | 汇总列在大纲中位于明细数据列的左侧。 |
| xlSummaryOnRight | -4152 | 汇总列在大纲中位于明细数据列的右侧。 |
