# XlTickMark 枚举

指定坐标轴的主要和次要刻度线标志的位置。

| 名称              | 值    | 说明         |
|-------------------|-------|--------------|
| xlTickMarkCross   | 4     | 跨坐标轴。   |
| xlTickMarkInside  | 2     | 在坐标轴内。 |
| xlTickMarkNone    | -4142 | 无刻度线。   |
| xlTickMarkOutside | 3     | 在坐标轴外。 |
