# XlTimePeriods 枚举

指定时间段。

| 名称        | 值  | 说明      |
|-------------|-----|-----------|
| xlLast7Days | 2   | 过去 7 天 |
| xlLastMonth | 5   | 上月      |
| xlLastWeek  | 4   | 上周      |
| xlNextMonth | 8   | 下月      |
| xlNextWeek  | 7   | 下周      |
| xlThisMonth | 9   | 本月      |
| xlThisWeek  | 3   | 本周      |
| xlToday     | 0   | 今天      |
| xlTomorrow  | 6   | 明天      |
| xlYesterday | 1   | 昨天      |
