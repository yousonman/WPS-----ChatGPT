# XlUpdateLinks 枚举

指定工作簿用于更新嵌入式 OLE 链接的设置。

| 名称                     | 值  | 说明                                              |
|--------------------------|-----|---------------------------------------------------|
| xlUpdateLinksAlways      | 3   | 始终更新指定工作簿的嵌入式 OLE 链接。             |
| xlUpdateLinksNever       | 2   | 从不更新指定工作簿的嵌入式 OLE 链接。             |
| xlUpdateLinksUserSetting | 1   | 按照用户对指定工作簿的设置，更新嵌入式 OLE 链接。 |
