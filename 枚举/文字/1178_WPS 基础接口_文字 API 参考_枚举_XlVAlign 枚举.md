# XlVAlign 枚举

指定对象的垂直对齐方式。

| 名称                | 值    | 说明       |
|---------------------|-------|------------|
| xlVAlignBottom      | -4107 | 底端对齐。 |
| xlVAlignCenter      | -4108 | 居中对齐。 |
| xlVAlignDistributed | -4117 | 分散对齐。 |
| xlVAlignJustify     | -4130 | 两端对齐。 |
| xlVAlignTop         | -4160 | 顶部对齐。 |
