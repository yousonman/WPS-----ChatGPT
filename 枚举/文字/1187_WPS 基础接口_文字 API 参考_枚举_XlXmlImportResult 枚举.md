# XlXmlImportResult 枚举

指定刷新或导入操作的结果。

| 名称                         | 值  | 说明                                                            |
|------------------------------|-----|-----------------------------------------------------------------|
| xlXmlImportElementsTruncated | 1   | 由于指定的 XML 数据文件对于工作表来说太大，因此其内容已被截断。 |
| xlXmlImportSuccess           | 0   | XML 数据文件已成功导入。                                        |
| xlXmlImportValidationFailed  | 2   | XML 数据文件的内容不符合指定的架构映射。                        |
