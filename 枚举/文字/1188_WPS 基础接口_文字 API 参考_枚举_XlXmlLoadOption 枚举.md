# XlXmlLoadOption 枚举

指定 ET 打开 XML 数据文件的方式。

| 名称                  | 值  | 说明                                              |
|-----------------------|-----|---------------------------------------------------|
| xlXmlLoadImportToList | 2   | 将 XML 数据文件的内容置于 XML 表中。              |
| xlXmlLoadMapXml       | 3   | 在 “XML 结构” 任务窗格中显示 XML 数据文件的架构。 |
| xlXmlLoadOpenXml      | 1   | 打开 XML 数据文件。文件的内容将展开。             |
| xlXmlLoadPromptUser   | 0   | 提示用户选择打开文件的方式。                      |
