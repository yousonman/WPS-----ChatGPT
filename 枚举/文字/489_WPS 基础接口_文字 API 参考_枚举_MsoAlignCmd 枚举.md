# MsoAlignCmd 枚举

定义指定对象相互之间的对齐方式。

| 名称            | 值  | 说明               |
|-----------------|-----|--------------------|
| msoAlignBottoms | 5   | 指定对象底端对齐。 |
| msoAlignCenters | 1   | 指定对象中心对齐。 |
| msoAlignLefts   | 0   | 指定对象左侧对齐。 |
| msoAlignMiddles | 4   | 指定对象中部对齐。 |
| msoAlignRights  | 2   | 指定对象右侧对齐。 |
| msoAlignTops    | 3   | 指定对象顶端对齐。 |
