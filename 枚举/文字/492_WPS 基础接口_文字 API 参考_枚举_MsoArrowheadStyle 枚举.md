# MsoArrowheadStyle 枚举

指定线条末端的箭头样式。

| 名称                   | 值  | 说明                           |
|------------------------|-----|--------------------------------|
| msoArrowheadDiamond    | 5   | 菱形。                         |
| msoArrowheadNone       | 1   | 无箭头。                       |
| msoArrowheadOpen       | 3   | 开放。                         |
| msoArrowheadOval       | 6   | 椭圆形。                       |
| msoArrowheadStealth    | 4   | 燕尾形。                       |
| msoArrowheadStyleMixed | -2  | 只返回值，表示其他状态的组合。 |
| msoArrowheadTriangle   | 2   | 三角形。                       |
