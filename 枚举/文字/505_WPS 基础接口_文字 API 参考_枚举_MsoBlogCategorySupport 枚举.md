# MsoBlogCategorySupport 枚举

指定提供程序支持多少个类别。

| 名称                      | 值  | 说明             |
|---------------------------|-----|------------------|
| msoBlogMultipleCategories | 2   | 支持多个类别。   |
| msoBlogNoCategories       | 0   | 不支持任何类别。 |
| msoBlogOneCategory        | 1   | 支持一个类别。   |
