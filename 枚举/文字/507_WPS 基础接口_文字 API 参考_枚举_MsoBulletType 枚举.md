# MsoBulletType 枚举

指定项目符号类型。

| 名称                | 值  | 说明                           |
|---------------------|-----|--------------------------------|
| msoBulletMixed      | -2  | 只返回值，表示其他状态的组合。 |
| msoBulletNone       | 0   | 无项目符号。                   |
| msoBulletNumbered   | 2   | 编号项目符号。                 |
| msoBulletPicture    | 3   | 图片项目符号。                 |
| msoBulletUnnumbered | 1   | 非编号项目符号。               |
