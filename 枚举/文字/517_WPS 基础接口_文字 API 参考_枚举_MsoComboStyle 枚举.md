# MsoComboStyle 枚举

指定命令栏组合框是否包括标签。

| 名称           | 值  | 说明                                          |
|----------------|-----|-----------------------------------------------|
| msoComboLabel  | 1   | 组合框包括由组合框的 Caption 属性指定的标签。 |
| msoComboNormal | 0   | 组合框不包括标签。                            |
