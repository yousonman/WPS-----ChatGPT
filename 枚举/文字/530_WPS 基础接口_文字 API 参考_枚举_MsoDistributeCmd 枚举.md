# MsoDistributeCmd 枚举

指定如何平均分布图形集合。与 **ShapeRange** 集合的 **Distribute** 方法一起使用。

| 名称                      | 值  | 说明       |
|---------------------------|-----|------------|
| msoDistributeHorizontally | 0   | 水平分布。 |
| msoDistributeVertically   | 1   | 垂直分布。 |
