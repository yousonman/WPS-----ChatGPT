# MsoEditingType 枚举

指定节点的编辑类型。

| 名称                | 值  | 说明                           |
|---------------------|-----|--------------------------------|
| msoEditingAuto      | 0   | 编辑类型适合于正在连接的线段。 |
| msoEditingCorner    | 1   | 角部节点。                     |
| msoEditingSmooth    | 2   | 平滑结点。                     |
| msoEditingSymmetric | 3   | 对称节点。                     |
