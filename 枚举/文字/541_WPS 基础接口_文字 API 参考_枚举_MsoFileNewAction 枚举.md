# MsoFileNewAction 枚举

指定在用户单击任务窗格中的项时所执行的操作。与 **NewFile** 对象的 **Add** 方法一起使用。

| 名称             | 值  | 说明       |
|------------------|-----|------------|
| msoCreateNewFile | 1   | 新建文件。 |
| msoEditFile      | 0   | 编辑文件。 |
| msoOpenFile      | 2   | 打开文件。 |
