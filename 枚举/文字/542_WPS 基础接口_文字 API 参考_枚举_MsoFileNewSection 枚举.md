# MsoFileNewSection 枚举

指定要向其中添加文件或文件引用所在的任务窗格部分。与 **NewFile** 对象的 **Add** 方法和 **Remove** 方法一起使用。

| 名称                   | 值  | 说明                      |
|------------------------|-----|---------------------------|
| msoBottomSection       | 4   | “底部” 部分。             |
| msoNew                 | 1   | “新建” 部分。             |
| msoNewfromExistingFile | 2   | “根据现有文件新建” 部分。 |
| msoNewfromTemplate     | 3   | “根据模板新建” 部分。     |
| msoOpenDocument        | 0   | “打开文档” 部分。         |
