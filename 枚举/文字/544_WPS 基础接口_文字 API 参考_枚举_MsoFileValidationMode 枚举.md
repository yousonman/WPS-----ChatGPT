# MsoFileValidationMode 枚举

指定文件验证模式。

| 名称                     | 值  | 说明               |
|--------------------------|-----|--------------------|
| msoFileValidationDefault | 0   | 验证文件（默认）。 |
| msoFileValidationSkip    | 1   | 不验证文件。       |
