# MsoFilterConjunction 枚举

指定一个筛选条件与其他筛选条件相关联的方式。与 **ODSOFilters** 对象的 **Conjunction** 属性以及 **MailMergeFilters** 对象的 **Add** 方法一起使用。

| 名称                    | 值  | 说明     |
|-------------------------|-----|----------|
| msoFilterConjunctionAnd | 0   | 与关联。 |
| msoFilterConjunctionOr  | 1   | 或关联。 |
