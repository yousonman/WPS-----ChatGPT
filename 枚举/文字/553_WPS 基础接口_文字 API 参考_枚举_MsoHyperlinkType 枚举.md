# MsoHyperlinkType 枚举

指定超链接的类型。

| 名称                    | 值  | 说明                               |
|-------------------------|-----|------------------------------------|
| msoHyperlinkInlineShape | 2   | 超链接应用于内嵌形状。仅用于 WPS。 |
| msoHyperlinkRange       | 0   | 超链接应用于 Range 对象。          |
| msoHyperlinkShape       | 1   | 超链接应用于 Shape 对象。          |
