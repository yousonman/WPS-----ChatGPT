# MsoPickerField 枚举

指定在文件对话框中为用户显示的视图。

| 名称                   | 值  | 说明           |
|------------------------|-----|----------------|
| msoPickerFieldUnknown  | 0   | 未知字段。     |
| msoPickerFieldDateTime | 1   | 日期字段。     |
| msoPickerFieldNumber   | 2   | 数字字段。     |
| msoPickerFieldText     | 3   | 文本字段。     |
| msoPickerFieldUser     | 4   | 用户或组字段。 |
| msoPickerFieldMax      | 5   | 列举的警示值。 |

> WPS 开发文档： [WPS 基础接口/文字 API 参考/枚举/MsoPickerField 枚举](https://qn.cache.wpscdn.cn/encs/doc/office_v19/topics/WPS%20%E5%9F%BA%E7%A1%80%E6%8E%A5%E5%8F%A3/%E6%96%87%E5%AD%97%20API%20%E5%8F%82%E8%80%83/%E6%9E%9A%E4%B8%BE/MsoPickerField%20%E6%9E%9A%E4%B8%BE.html)
