# MsoPresetLightingSoftness 枚举

指定形状上所使用光的亮度。

| 名称                           | 值  | 说明     |
|--------------------------------|-----|----------|
| msoLightingBright              | 3   | 明亮光。 |
| msoLightingDim                 | 1   | 暗淡光。 |
| msoLightingNormal              | 2   | 普通光。 |
| msoPresetLightingSoftnessMixed | -2  | 不支持。 |
