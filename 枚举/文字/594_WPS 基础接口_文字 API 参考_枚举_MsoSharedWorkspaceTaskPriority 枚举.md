# MsoSharedWorkspaceTaskPriority 枚举

指定共享工作区任务的优先级。

| 名称                                 | 值  | 说明         |
|--------------------------------------|-----|--------------|
| msoSharedWorkspaceTaskPriorityHigh   | 1   | 高优先级。   |
| msoSharedWorkspaceTaskPriorityLow    | 3   | 低优先级。   |
| msoSharedWorkspaceTaskPriorityNormal | 2   | 普通优先级。 |
