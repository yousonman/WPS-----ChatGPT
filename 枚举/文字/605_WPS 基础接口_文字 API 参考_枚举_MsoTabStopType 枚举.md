# MsoTabStopType 枚举

指定制表位的类型。

| 名称              | 值  | 说明                           |
|-------------------|-----|--------------------------------|
| msoTabStopCenter  | 2   | 中央制表位。                   |
| msoTabStopDecimal | 4   | 小数制表位。                   |
| msoTabStopLeft    | 1   | 左制表位。                     |
| msoTabStopMixed   | -2  | 只返回值，表示其他状态的组合。 |
| msoTabStopRight   | 3   | 右制表位。                     |
