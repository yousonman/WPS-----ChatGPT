# MsoTextDirection 枚举

指定文本行文方向。

| 名称                        | 值  | 说明                           |
|-----------------------------|-----|--------------------------------|
| msoTextDirectionLeftToRight | 1   | 文本左至右行文。               |
| msoTextDirectionMixed       | -2  | 只返回值，表示其他状态的组合。 |
| msoTextDirectionRightToLeft | 2   | 文本右至左行文。               |
