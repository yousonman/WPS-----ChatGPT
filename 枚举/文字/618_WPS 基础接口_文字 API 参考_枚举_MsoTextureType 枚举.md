# MsoTextureType 枚举

指定选定填充的纹理类型。

| 名称                  | 值  | 说明                           |
|-----------------------|-----|--------------------------------|
| msoTexturePreset      | 1   | 预设纹理类型。                 |
| msoTextureTypeMixed   | -2  | 只返回值，表示其他状态的组合。 |
| msoTextureUserDefined | 2   | 用户定义的纹理类型。           |
