# RibbonControlSize 枚举

指定一些常量，用于定义功能区中的控件大小。

| 名称                     | 值  | 说明     |
|--------------------------|-----|----------|
| RibbonControlSizeLarge   | 1   | 大控件。 |
| RibbonControlSizeRegular | 0   | 小控件。 |
