# WdAlertLevel 枚举

指定运行宏时产生的一些警告和消息的处理方式。

| 名称               | 值  | 说明                                                           |
|--------------------|-----|----------------------------------------------------------------|
| wdAlertsAll        | -1  | 显示所有的消息框和警告，并将错误返回宏。                       |
| wdAlertsMessageBox | -2  | 仅显示消息框，同时捕获错误并将其返回宏。                       |
| wdAlertsNone       | 0   | 不显示警告或消息框。如果出现消息框，宏将选择默认值并继续运行。 |
