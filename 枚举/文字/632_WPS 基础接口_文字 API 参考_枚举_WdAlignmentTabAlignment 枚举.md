# WdAlignmentTabAlignment 枚举

指定制表符的对齐方式。

| 名称     | 值  | 说明             |
|----------|-----|------------------|
| wdCenter | 1   | 居中的制表符。   |
| wdLeft   | 0   | 左对齐的制表符。 |
| wdRight  | 2   | 右对齐的制表符。 |
