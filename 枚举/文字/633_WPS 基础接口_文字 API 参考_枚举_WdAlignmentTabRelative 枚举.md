# WdAlignmentTabRelative 枚举

指定 WPS 在计算制表位的对齐方式时是相对于节边距还是相对于段落缩进。

| 名称     | 值  | 说明                                     |
|----------|-----|------------------------------------------|
| wdIndent | 1   | WPS 相对于段落缩进计算制表位的对齐方式。 |
| wdMargin | 0   | WPS 相对于边距计算制表位的对齐方式。     |
