# WdAutoMacros 枚举

| 名称        | 值  | 说明           |
|-------------|-----|----------------|
| wdAutoClose | 3   | AutoClose 宏。 |
| wdAutoExec  | 0   | AutoExec 宏。  |
| wdAutoExit  | 4   | AutoExit 宏。  |
| wdAutoNew   | 1   | AutoNew 宏。   |
| wdAutoOpen  | 2   | AutoOpen 宏。  |
| wdAutoSync  | 5   | AutoSync 宏。  |
