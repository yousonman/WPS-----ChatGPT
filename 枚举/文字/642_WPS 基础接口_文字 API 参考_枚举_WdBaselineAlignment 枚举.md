# WdBaselineAlignment 枚举

指定行中字体的垂直位置。

| 名称                     | 值  | 说明                             |
|--------------------------|-----|----------------------------------|
| wdBaselineAlignAuto      | 4   | WPS 自动调整字体的基线对齐方式。 |
| wdBaselineAlignBaseline  | 2   | 对齐段落的基线。                 |
| wdBaselineAlignCenter    | 1   | 对齐每个字体的中心点。           |
| wdBaselineAlignFarEast50 | 3   | 按亚洲语言字体标准的基线对齐。   |
| wdBaselineAlignTop       | 0   | 沿每个字体顶端对齐。             |
