# WdBookmarkSortBy 枚举

为显示在 **“书签”** 对话框中的书签名称指定排序选项。

| 名称             | 值  | 说明                   |
|------------------|-----|------------------------|
| wdSortByLocation | 1   | 按在文档中的位置排序。 |
| wdSortByName     | 0   | 按书签名称排序。       |
