# WdBorderType 枚举

指定要使用的边框类型。

| 名称                 | 值  | 说明                           |
|----------------------|-----|--------------------------------|
| wdBorderBottom       | -3  | 底边框线。                     |
| wdBorderDiagonalDown | -7  | 方向从左上角开始的斜向边框线。 |
| wdBorderDiagonalUp   | -8  | 方向从左下角开始的斜向边框线。 |
| wdBorderHorizontal   | -5  | 横向框线。                     |
| wdBorderLeft         | -2  | 左侧框线。                     |
| wdBorderRight        | -4  | 右侧框线。                     |
| wdBorderTop          | -1  | 上框线。                       |
| wdBorderVertical     | -6  | 纵向框线。                     |
