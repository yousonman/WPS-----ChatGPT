# WdCellVerticalAlignment 枚举

指定表格中一个或多个单元格的文字的垂直对齐方式。

| 名称                      | 值  | 说明                       |
|---------------------------|-----|----------------------------|
| wdCellAlignVerticalBottom | 3   | 文字与单元格底边框线对齐。 |
| wdCellAlignVerticalCenter | 1   | 文字与单元格中心对齐。     |
| wdCellAlignVerticalTop    | 0   | 文字与单元格上框线对齐。   |
