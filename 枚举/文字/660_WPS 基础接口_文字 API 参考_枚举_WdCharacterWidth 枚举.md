# WdCharacterWidth 枚举

为指定范围内的字符指定字符宽度。

| 名称             | 值  | 说明                     |
|------------------|-----|--------------------------|
| wdWidthFullWidth | 7   | 以完整字符宽度显示字符。 |
| wdWidthHalfWidth | 6   | 以半个字符宽度显示字符。 |
