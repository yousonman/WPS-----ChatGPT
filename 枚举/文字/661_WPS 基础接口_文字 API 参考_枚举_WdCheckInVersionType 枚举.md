# WdCheckInVersionType 枚举

指定在使用 **CheckIn** 方法时签入文档的版本类型。适用于存储在 SharePoint 库中的文档。

| 名称                      | 值  | 说明                     |
|---------------------------|-----|--------------------------|
| wdCheckInMajorVersion     | 1   | 主版本。                 |
| wdCheckInMinorVersion     | 0   | 次版本。                 |
| wdCheckInOverwriteVersion | 2   | 覆盖服务器上的当前版本。 |
