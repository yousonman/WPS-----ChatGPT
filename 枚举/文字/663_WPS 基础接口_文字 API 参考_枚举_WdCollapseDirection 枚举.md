# WdCollapseDirection 枚举

指定某范围或所选内容的折叠方向。

| 名称            | 值  | 说明                   |
|-----------------|-----|------------------------|
| wdCollapseEnd   | 0   | 将该范围折叠到结束点。 |
| wdCollapseStart | 1   | 将该范围折叠到起始点。 |
