# WdContinue 枚举

指定是否可以继续使用上一列表中的格式。

| 名称               | 值  | 说明                           |
|--------------------|-----|--------------------------------|
| wdContinueDisabled | 0   | 不能继续使用上一列表中的格式。 |
| wdContinueList     | 2   | 可以继续使用上一列表中的格式。 |
| wdResetList        | 1   | 可以重新开始编号。             |
