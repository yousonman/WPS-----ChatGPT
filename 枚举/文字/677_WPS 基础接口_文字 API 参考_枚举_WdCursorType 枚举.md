# WdCursorType 枚举

指定光标的状态（形状）。

| 名称                   | 值  | 说明                                                                    |
|------------------------|-----|-------------------------------------------------------------------------|
| wdCursorIBeam          | 1   | I 形光标形状。                                                          |
| wdCursorNormal         | 2   | 正常光标形状。默认值；光标采用 Microsoft Windows 或应用程序指定的形状。 |
| wdCursorNorthwestArrow | 3   | 方向从左上角开始的斜线光标形状。                                        |
| wdCursorWait           | 0   | 沙漏光标形状。                                                          |
