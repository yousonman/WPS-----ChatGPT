# WdDateLanguage 枚举

指定要应用于当前日期和/或时间的语言格式。

| 名称                | 值   | 说明                  |
|---------------------|------|-----------------------|
| wdDateLanguageBidi  | 10   | 双向的日期/时间格式。 |
| wdDateLanguageLatin | 1033 | 拉丁文日期/时间格式。 |
