# WdDefaultTableBehavior 枚举

指定 WPS 是否要根据表格单元格的内容自动调整其大小（“自动调整”功能）。

| 名称                 | 值  | 说明                           |
|----------------------|-----|--------------------------------|
| wdWord8TableBehavior | 0   | 禁用“自动调整”功能，默认设置。 |
| wdWord9TableBehavior | 1   | 启用“自动调整”功能。           |
