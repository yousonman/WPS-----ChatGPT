# WdDiacriticColor 枚举

指定是否向双向语言或拉丁样式语言的音调符号应用不同的颜色。

| 名称                  | 值  | 说明                               |
|-----------------------|-----|------------------------------------|
| wdDiacriticColorBidi  | 0   | 双向语言（阿拉伯语、希伯来语等）。 |
| wdDiacriticColorLatin | 1   | 拉丁样式的语言。                   |
