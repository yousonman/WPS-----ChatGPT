# WdDisableFeaturesIntroducedAfter 枚举

指定 WPS 版本，禁用该版本之后引入的所有功能。

| 名称   | 值  | 说明                                             |
|--------|-----|--------------------------------------------------|
| wd70   | 0   | 指定 WPS for Windows 的 7.0 和 7.0a 版。         |
| wd70FE | 1   | 指定 WPS for Windows 的 7.0 和 7.0a 版，亚洲版。 |
| wd80   | 2   | 指定 WPS 97 for Windows，默认值。                |
