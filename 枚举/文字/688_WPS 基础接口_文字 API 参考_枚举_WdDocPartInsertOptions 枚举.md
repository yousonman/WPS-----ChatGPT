# WdDocPartInsertOptions 枚举

指定如何将生成块插入文档。

| 名称              | 值  | 说明             |
|-------------------|-----|------------------|
| wdInsertContent   | 0   | 嵌入型生成块。   |
| wdInsertPage      | 2   | 页面级别生成块。 |
| wdInsertParagraph | 1   | 段落级别生成块。 |
