# WdDocumentKind 枚举

指定 WPS 自动设置文档格式时所使用的格式类型。

| 名称                   | 值  | 说明           |
|------------------------|-----|----------------|
| wdDocumentEmail        | 2   | 电子邮件格式。 |
| wdDocumentLetter       | 1   | 信函格式。     |
| wdDocumentNotSpecified | 0   | 无指定格式。   |
