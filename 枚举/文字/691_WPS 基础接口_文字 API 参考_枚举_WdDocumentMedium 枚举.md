# WdDocumentMedium 枚举

指定要应用主题的文档类型。

| 名称           | 值  | 说明       |
|----------------|-----|------------|
| wdDocument     | 1   | 文档。     |
| wdEmailMessage | 0   | 电子邮件。 |
| wdWebPage      | 2   | 网页。     |
