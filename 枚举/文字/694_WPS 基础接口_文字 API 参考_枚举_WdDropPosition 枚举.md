# WdDropPosition 枚举

指定首字下沉的位置。

| 名称         | 值  | 说明                   |
|--------------|-----|------------------------|
| wdDropMargin | 2   | 首字下沉在左边距结束。 |
| wdDropNone   | 0   | 没有首字下沉。         |
| wdDropNormal | 1   | 首字下沉在左边距开始。 |
