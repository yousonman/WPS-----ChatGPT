# WdEnclosureType 枚举

指定要用作圈号的符号类型。

| 名称                | 值  | 说明     |
|---------------------|-----|----------|
| wdEnclosureCircle   | 0   | 圆圈。   |
| wdEnclosureDiamond  | 3   | 菱形。   |
| wdEnclosureSquare   | 1   | 方形。   |
| wdEnclosureTriangle | 2   | 三角形。 |
