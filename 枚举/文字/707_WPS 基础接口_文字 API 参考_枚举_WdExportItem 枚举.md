# WdExportItem 枚举

指定导出文档时是否带标记。

| 名称                       | 值  | 说明                 |
|----------------------------|-----|----------------------|
| wdExportDocumentContent    | 0   | 导出文档时不带标记。 |
| wdExportDocumentWithMarkup | 7   | 导出文档时带标记。   |
