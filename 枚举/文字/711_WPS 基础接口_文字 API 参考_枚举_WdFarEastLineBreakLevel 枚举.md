# WdFarEastLineBreakLevel 枚举

为所指定文档指定换行符控件级别。

| 名称                          | 值  | 说明               |
|-------------------------------|-----|--------------------|
| wdFarEastLineBreakLevelCustom | 2   | 自定义换行符控件。 |
| wdFarEastLineBreakLevelNormal | 0   | 普通换行符控件。   |
| wdFarEastLineBreakLevelStrict | 1   | 严格换行符控件。   |
