# WdFindWrap 枚举

指定在为查找操作指定某一所选内容或范围，但未在其中找到搜索文字时的换行行为。

| 名称           | 值  | 说明                                                                     |
|----------------|-----|--------------------------------------------------------------------------|
| wdFindAsk      | 2   | 搜索完所选内容或范围后，WPS 会显示一条消息，询问是否搜索文档的其余部分。 |
| wdFindContinue | 1   | 到达搜索范围的开始或结尾时，继续执行查找操作。                           |
| wdFindStop     | 0   | 到达搜索范围的开始或结尾时，停止执行查找操作。                           |
