# WdFootnoteLocation 枚举

指定脚注的位置。

| 名称           | 值  | 说明             |
|----------------|-----|------------------|
| wdBeneathText  | 1   | 在当前文本下方。 |
| wdBottomOfPage | 0   | 在当前页面底部。 |
