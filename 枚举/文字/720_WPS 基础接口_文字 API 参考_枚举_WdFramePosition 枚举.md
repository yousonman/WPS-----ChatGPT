# WdFramePosition 枚举

指定按其计算图文框位置的参考点。

| 名称           | 值      | 说明             |
|----------------|---------|------------------|
| wdFrameBottom  | -999997 | 下边距。         |
| wdFrameCenter  | -999995 | 文档中央。       |
| wdFrameInside  | -999994 | 图文框中的内容。 |
| wdFrameLeft    | -999998 | 左边距。         |
| wdFrameOutside | -999993 | 图文框外的内容。 |
| wdFrameRight   | -999996 | 右边距。         |
| wdFrameTop     | -999999 | 上边距。         |
