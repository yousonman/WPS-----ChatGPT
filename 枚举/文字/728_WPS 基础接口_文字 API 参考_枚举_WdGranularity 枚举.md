# WdGranularity 枚举

指定在比较两个文档时所要标记的修订的级别。

| 名称                   | 值  | 说明             |
|------------------------|-----|------------------|
| wdGranularityCharLevel | 0   | 在字符级别修订。 |
| wdGranularityWordLevel | 1   | 在字级别修订。   |
