# WdGutterStyle 枚举

指定装订线显示在文档中的位置。

| 名称             | 值  | 说明   |
|------------------|-----|--------|
| wdGutterPosLeft  | 0   | 左侧。 |
| wdGutterPosRight | 2   | 右侧。 |
| wdGutterPosTop   | 1   | 顶部。 |
