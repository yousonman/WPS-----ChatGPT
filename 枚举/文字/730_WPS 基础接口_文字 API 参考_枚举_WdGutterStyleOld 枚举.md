# WdGutterStyleOld 枚举

指定装订线样式应符合从左向右文字排列规则还是从右向左文字排列规则。

| 名称               | 值  | 说明                                               |
|--------------------|-----|----------------------------------------------------|
| wdGutterStyleBidi  | 2   | 应使用双向语言装订线才能符合从右向左文字排列规则。 |
| wdGutterStyleLatin | -10 | 应使用拉丁文装订线才能符合从左向右文字排列规则。   |
