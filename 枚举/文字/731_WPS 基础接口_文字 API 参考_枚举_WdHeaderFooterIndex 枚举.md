# WdHeaderFooterIndex 枚举

返回文档或节中的指定页眉或页脚。

| 名称                    | 值  | 说明                                           |
|-------------------------|-----|------------------------------------------------|
| wdHeaderFooterEvenPages | 3   | 返回偶数页上的所有页眉或页脚。                 |
| wdHeaderFooterFirstPage | 2   | 返回文档或节中的第一个页眉或页脚。             |
| wdHeaderFooterPrimary   | 1   | 返回文档或节中除第一页外所有页上的页眉或页脚。 |
