# WdHeadingSeparator 枚举

指定两个索引标题之间的文字分隔符类型。

| 名称                         | 值  | 说明             |
|------------------------------|-----|------------------|
| wdHeadingSeparatorBlankLine  | 1   | 空行。           |
| wdHeadingSeparatorLetter     | 2   | 指定的字母。     |
| wdHeadingSeparatorLetterFull | 4   | 指定的大写字母。 |
| wdHeadingSeparatorLetterLow  | 3   | 指定的小写字符。 |
| wdHeadingSeparatorNone       | 0   | 没有分隔符。     |
