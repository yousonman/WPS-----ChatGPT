# WdKana 枚举

指定所选日文文字的格式。

| 名称           | 值  | 说明                       |
|----------------|-----|----------------------------|
| wdKanaHiragana | 9   | 该文字的格式设置为平假名。 |
| wdKanaKatakana | 8   | 该文字的格式设置为片假名。 |
