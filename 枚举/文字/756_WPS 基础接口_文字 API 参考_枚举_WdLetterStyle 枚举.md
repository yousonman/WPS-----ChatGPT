# WdLetterStyle 枚举

指定“英文信函向导”所创建信函的布局。

| 名称            | 值  | 说明           |
|-----------------|-----|----------------|
| wdFullBlock     | 0   | 完整区域。     |
| wdModifiedBlock | 1   | 修改过的区域。 |
| wdSemiBlock     | 2   | 局部区域。     |
