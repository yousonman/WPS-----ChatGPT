# WdLineType 枚举

指定行是文本行还是表格行。

| 名称       | 值  | 说明                 |
|------------|-----|----------------------|
| wdTableRow | 1   | 表格行。             |
| wdTextLine | 0   | 文档正文中的文本行。 |
