# WdListApplyTo 枚举

指定要应用列表模板的列表部分。

| 名称                          | 值  | 说明                     |
|-------------------------------|-----|--------------------------|
| wdListApplyToSelection        | 2   | 所选内容。               |
| wdListApplyToThisPointForward | 1   | 从光标插入点到列表末尾。 |
| wdListApplyToWholeList        | 0   | 整个列表。               |
