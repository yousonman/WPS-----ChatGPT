# WdMailMergeDefaultRecord 枚举

指定在邮件合并过程中使用的默认记录。

| 名称                 | 值  | 说明                                   |
|----------------------|-----|----------------------------------------|
| wdDefaultFirstRecord | 1   | 将结果集内的第一条记录用作默认记录。   |
| wdDefaultLastRecord  | -16 | 将结果集内的最后一条记录用作默认记录。 |
