# WdMailMergeMailFormat 枚举

指定邮件合并目标为电子邮件时要使用的格式。

| 名称                  | 值  | 说明                                     |
|-----------------------|-----|------------------------------------------|
| wdMailFormatHTML      | 1   | 使用 HTML 格式发送邮件合并电子邮件文档。 |
| wdMailFormatPlainText | 0   | 使用纯文本格式发送邮件合并电子邮件文档。 |
