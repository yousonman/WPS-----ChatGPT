# WdMeasurementUnits 枚举

指定要使用的度量单位。

| 名称          | 值  | 说明                                       |
|---------------|-----|--------------------------------------------|
| wdCentimeters | 1   | 厘米。                                     |
| wdInches      | 0   | 英寸。                                     |
| wdMillimeters | 2   | 毫米。                                     |
| wdPicas       | 4   | 十二点活字（通常用于传统打字机字体间距）。 |
| wdPoints      | 3   | 磅。                                       |
