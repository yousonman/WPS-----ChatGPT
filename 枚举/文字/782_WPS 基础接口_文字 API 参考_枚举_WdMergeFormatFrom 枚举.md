# WdMergeFormatFrom 枚举

指定要在执行文档比较功能期间保持其格式的文档。

| 名称                      | 值  | 说明                             |
|---------------------------|-----|----------------------------------|
| wdMergeFormatFromOriginal | 0   | 保留原始文档的格式。             |
| wdMergeFormatFromPrompt   | 2   | 提示用户指定要保留其格式的文档。 |
| wdMergeFormatFromRevised  | 1   | 保留修订的文档中的格式。         |
