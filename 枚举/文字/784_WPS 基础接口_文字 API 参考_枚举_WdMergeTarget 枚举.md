# WdMergeTarget 枚举

指定比较和合并文档过程的目标。

| 名称                  | 值  | 说明               |
|-----------------------|-----|--------------------|
| wdMergeTargetCurrent  | 1   | 合并至当前文档中。 |
| wdMergeTargetNew      | 2   | 合并至新文档中。   |
| wdMergeTargetSelected | 0   | 合并至所选文档中。 |
