# WdMovementType 枚举

指定移动所选内容的方式。

| 名称     | 值  | 说明                                                   |
|----------|-----|--------------------------------------------------------|
| wdExtend | 1   | 将所选内容的末尾扩展到指定单位的末尾。                 |
| wdMove   | 0   | 将所选内容折叠到插入点中并移到指定单位的末尾。默认值。 |
