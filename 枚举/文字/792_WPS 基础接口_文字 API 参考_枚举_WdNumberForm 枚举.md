# WdNumberForm 枚举

指定 OpenType 字体的数字形式设置。

| 名称                 | 值  | 说明                         |
|----------------------|-----|------------------------------|
| wdNumberFormDefault  | 0   | 应用字体的默认数字形式。     |
| wdNumberFormLining   | 1   | 对字体应用内衬数字形式。     |
| wdNumberFormOldstyle | 2   | 对字体应用“旧样式”数字形式。 |
