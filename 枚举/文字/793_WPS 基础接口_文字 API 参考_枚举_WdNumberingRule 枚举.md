# WdNumberingRule 枚举

指定要应用的编号规则。

| 名称                | 值  | 说明               |
|---------------------|-----|--------------------|
| wdRestartContinuous | 0   | 连续分配编号。     |
| wdRestartPage       | 2   | 重设每一页的页码。 |
| wdRestartSection    | 1   | 重设每一节的节号。 |
