# WdNumberSpacing 枚举

指定 OpenType 字体的数字间距设置。

| 名称                        | 值  | 说明                       |
|-----------------------------|-----|----------------------------|
| wdNumberSpacingDefault      | 0   | 应用字体的默认数字间距。   |
| wdNumberSpacingProportional | 1   | 对字体应用成比例数字间距。 |
| wdNumberSpacingTabular      | 2   | 对字体应用表格数字间距。   |
