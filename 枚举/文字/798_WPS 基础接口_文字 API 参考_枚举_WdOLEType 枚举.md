# WdOLEType 枚举

指定 OLE 对象的类型。

| 名称         | 值  | 说明              |
|--------------|-----|-------------------|
| wdOLEControl | 2   | OLE 控件。        |
| wdOLEEmbed   | 1   | 嵌入的 OLE 对象。 |
| wdOLELink    | 0   | 链接的 OLE 对象。 |
