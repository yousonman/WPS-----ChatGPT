# WdOMathJc 枚举

指定公式默认的对齐方式。

| 名称                 | 值  | 说明             |
|----------------------|-----|------------------|
| wdOMathJcCenter      | 2   | 居中。           |
| wdOMathJcCenterGroup | 1   | 作为一个组居中。 |
| wdOMathJcInline      | 7   | 内嵌。           |
| wdOMathJcLeft        | 3   | 左对齐。         |
| wdOMathJcRight       | 4   | 右对齐。         |
