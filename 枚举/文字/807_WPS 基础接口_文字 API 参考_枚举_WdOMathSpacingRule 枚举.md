# WdOMathSpacingRule 枚举

| wdOMathSpacingDouble   | 2   | 双倍行距。       |
| wdOMathSpacingExactly  | 3   | 固定行距度量。   |
| wdOMathSpacingMultiple | 4   | 自定义行距度量。 |
| wdOMathSpacingSingle   | 0   | 单倍行距。       |
