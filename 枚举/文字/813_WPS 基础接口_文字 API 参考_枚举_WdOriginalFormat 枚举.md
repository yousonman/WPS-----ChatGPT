# WdOriginalFormat 枚举

指定文档格式。在保存文档时通常使用此枚举。

| 名称                     | 值  | 说明                   |
|--------------------------|-----|------------------------|
| wdOriginalDocumentFormat | 1   | 原始文档格式。         |
| wdPromptUser             | 2   | 提示用户选择文档格式。 |
| wdWordDocument           | 0   | WPS 文档格式。         |
