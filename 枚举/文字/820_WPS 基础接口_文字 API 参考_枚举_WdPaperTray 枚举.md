# WdPaperTray 枚举

指定打印机纸盒。

| 名称                        | 值  | 说明               |
|-----------------------------|-----|--------------------|
| wdPrinterAutomaticSheetFeed | 7   | 自动送纸。         |
| wdPrinterDefaultBin         | 0   | 默认纸盒。         |
| wdPrinterEnvelopeFeed       | 5   | 信封送纸器。       |
| wdPrinterFormSource         | 15  | 表单来源。         |
| wdPrinterLargeCapacityBin   | 11  | 大容量纸盒。       |
| wdPrinterLargeFormatBin     | 10  | 大格式纸盒。       |
| wdPrinterLowerBin           | 2   | 下层纸盒。         |
| wdPrinterManualEnvelopeFeed | 6   | 手动信封送纸。     |
| wdPrinterManualFeed         | 4   | 手动送纸。         |
| wdPrinterMiddleBin          | 3   | 中层纸盒。         |
| wdPrinterOnlyBin            | 1   | 打印机的唯一纸盒。 |
| wdPrinterPaperCassette      | 14  | 磁带式纸盒。       |
| wdPrinterSmallFormatBin     | 9   | 小格式纸盒。       |
| wdPrinterTractorFeed        | 8   | 送纸器。           |
| wdPrinterUpperBin           | 1   | 上层纸盒。         |
