# WdPartOfSpeech 枚举

指定某个单词在由 WPS 同义词库服务返回时所代表的词性。

| 名称           | 值  | 说明           |
|----------------|-----|----------------|
| wdAdjective    | 0   | 形容词。       |
| wdAdverb       | 2   | 副词。         |
| wdConjunction  | 5   | 连词。         |
| wdIdiom        | 8   | 成语。         |
| wdInterjection | 7   | 感叹词。       |
| wdNoun         | 1   | 名词。         |
| wdOther        | 9   | 其他某个词性。 |
| wdPreposition  | 6   | 介词。         |
| wdPronoun      | 4   | 代词。         |
| wdVerb         | 3   | 动词。         |
