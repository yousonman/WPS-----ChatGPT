# WdPreferredWidthType 枚举

指定要在测量项目宽度时使用的首选度量单位。

| 名称                    | 值  | 说明                                       |
|-------------------------|-----|--------------------------------------------|
| wdPreferredWidthAuto    | 1   | 基于当前所选内容自动选择要使用的度量单位。 |
| wdPreferredWidthPercent | 2   | 使用指定的百分比测量当前项目的宽度。       |
| wdPreferredWidthPoints  | 3   | 使用指定的磅数测量当前项目的宽度。         |
