# WdProtectionType 枚举

指定应用于当前文档的保护的类型。

| 名称                  | 值  | 说明                                 |
|-----------------------|-----|--------------------------------------|
| wdAllowOnlyComments   | 1   | 只允许将批注添加到文档中。           |
| wdAllowOnlyFormFields | 2   | 只允许通过窗体域将内容添加到文档中。 |
| wdAllowOnlyReading    | 3   | 允许对文档进行只读访问。             |
| wdAllowOnlyRevisions  | 0   | 只允许对现有内容进行修订。           |
| wdNoProtection        | -1  | 不对文档应用保护。                   |
