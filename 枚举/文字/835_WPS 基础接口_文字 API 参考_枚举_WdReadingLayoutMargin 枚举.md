# WdReadingLayoutMargin 枚举

指定以全屏阅读视图查看文档时边距可见还是隐藏。

| 名称              | 值  | 说明                                             |
|-------------------|-----|--------------------------------------------------|
| wdAutomaticMargin | 0   | 显示页面时不带边距。                             |
| wdFullMargin      | 2   | 显示页面时带边距。                               |
| wdSuppressMargin  | 1   | WPS 会自动根据可用空间来决定是显示还是隐藏边距。 |
