# WdReadingOrder 枚举

指定文档的文字排列和读取顺序。

| 名称              | 值  | 说明                 |
|-------------------|-----|----------------------|
| wdReadingOrderLtr | 1   | 从左到右的读取顺序。 |
| wdReadingOrderRtl | 0   | 从右到左的读取顺序。 |
