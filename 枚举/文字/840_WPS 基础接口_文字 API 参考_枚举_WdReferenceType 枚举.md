# WdReferenceType 枚举

指定要交叉引用的项目的类型。

| 名称                  | 值  | 说明     |
|-----------------------|-----|----------|
| wdRefTypeBookmark     | 2   | 书签。   |
| wdRefTypeEndnote      | 4   | 尾注。   |
| wdRefTypeFootnote     | 3   | 脚注。   |
| wdRefTypeHeading      | 1   | 标题。   |
| wdRefTypeNumberedItem | 0   | 编号项。 |
