# WdReplace 枚举

指定在使用“查找和替换”时要进行的替换次数。

| 名称          | 值  | 说明                     |
|---------------|-----|--------------------------|
| wdReplaceAll  | 2   | 替换所有匹配项。         |
| wdReplaceNone | 0   | 不替换任何匹配项。       |
| wdReplaceOne  | 1   | 替换遇到的第一个匹配项。 |
