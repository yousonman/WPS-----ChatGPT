# WdRevisedLinesMark 枚举

指定在启用修订功能时显示修订的位置。

| 名称                            | 值  | 说明         |
|---------------------------------|-----|--------------|
| wdRevisedLinesMarkLeftBorder    | 1   | 在左边框中。 |
| wdRevisedLinesMarkNone          | 0   | 不显示。     |
| wdRevisedLinesMarkOutsideBorder | 3   | 在边框外部。 |
| wdRevisedLinesMarkRightBorder   | 2   | 在右边框中。 |
