# WdRevisionsBalloonWidthType 枚举

指定 WPS 度量修订气球宽度的方式。

| 名称                  | 值  | 说明                     |
|-----------------------|-----|--------------------------|
| wdBalloonWidthPercent | 0   | 以文档宽度的百分比度量。 |
| wdBalloonWidthPoints  | 1   | 以磅为单位度量。         |
