# WdRevisionsWrap 枚举

指定在文档中显示修订标记的方式。

| 名称         | 值  | 说明                                 |
|--------------|-----|--------------------------------------|
| wdWrapAlways | 1   | 以环绕方式显示修订。                 |
| wdWrapAsk    | 2   | 询问用户是否应该以环绕方式显示修订。 |
| wdWrapNever  | 0   | 从不以环绕方式显示修订。             |
