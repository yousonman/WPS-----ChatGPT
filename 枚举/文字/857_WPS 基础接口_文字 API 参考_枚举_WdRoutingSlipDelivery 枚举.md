# WdRoutingSlipDelivery 枚举

指定用于将文档传递给收件人的传送方法。

| 名称              | 值  | 说明                     |
|-------------------|-----|--------------------------|
| wdAllAtOnce       | 1   | 同时传递给所有收件人。   |
| wdOneAfterAnother | 0   | 一次只传递给一位收件人。 |
