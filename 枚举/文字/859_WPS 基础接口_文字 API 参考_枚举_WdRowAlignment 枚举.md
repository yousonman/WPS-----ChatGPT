# WdRowAlignment 枚举

指定所选行中内容的对齐方式。

| 名称             | 值  | 说明             |
|------------------|-----|------------------|
| wdAlignRowCenter | 1   | 居中。           |
| wdAlignRowLeft   | 0   | 左对齐。默认值。 |
| wdAlignRowRight  | 2   | 右对齐。         |
