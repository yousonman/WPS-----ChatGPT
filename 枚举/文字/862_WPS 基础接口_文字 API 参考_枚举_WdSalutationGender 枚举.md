# WdSalutationGender 枚举

指定要在称呼中使用的性别。

| 名称            | 值  | 说明       |
|-----------------|-----|------------|
| wdGenderFemale  | 0   | 女性。     |
| wdGenderMale    | 1   | 男性。     |
| wdGenderNeutral | 2   | 中性。     |
| wdGenderUnknown | 3   | 未知性别。 |
