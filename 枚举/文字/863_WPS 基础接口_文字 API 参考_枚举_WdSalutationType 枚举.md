# WdSalutationType 枚举

指定要用于信函的称呼的类型。

| 名称                 | 值  | 说明         |
|----------------------|-----|--------------|
| wdSalutationBusiness | 2   | 商业称呼。   |
| wdSalutationFormal   | 1   | 正式称呼。   |
| wdSalutationInformal | 0   | 非正式称呼。 |
| wdSalutationOther    | 3   | 自定义称呼。 |
