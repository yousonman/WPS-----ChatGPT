# WdSelectionFlags 枚举

指定所选内容的属性。

| 名称             | 值  | 说明                         |
|------------------|-----|------------------------------|
| wdSelActive      | 8   | 所选内容是活动的。           |
| wdSelAtEOL       | 2   | 所选内容位于信函的末尾。     |
| wdSelOvertype    | 4   | 所选内容被改写。             |
| wdSelReplace     | 16  | 所选内容被替换。             |
| wdSelStartActive | 1   | 所选内容位于活动文档的开头。 |
