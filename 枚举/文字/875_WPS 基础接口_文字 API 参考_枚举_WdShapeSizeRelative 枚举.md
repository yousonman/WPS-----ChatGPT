# WdShapeSizeRelative 枚举

指定 **Shape** 或 **ShapeRange** 对象的 **WidthRelative** 或 **HeightRelative** 属性是否有效。

| 名称                    | 值      | 说明                                                                                                 |
|-------------------------|---------|------------------------------------------------------------------------------------------------------|
| wdShapeSizeRelativeNone | -999999 | 指定 WidthRelative 或 HeightRelative 属性当前无效，因此分别根据 Left 或 Top 属性中指定的值放置形状。 |
