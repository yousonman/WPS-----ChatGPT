# WdSortFieldType 枚举

指定要在对列进行排序时应用的排序类型。

| 名称                    | 值  | 说明             |
|-------------------------|-----|------------------|
| wdSortFieldAlphanumeric | 0   | 字母数字顺序。   |
| wdSortFieldDate         | 2   | 日期顺序。       |
| wdSortFieldJapanJIS     | 4   | 日语 JIS 顺序。  |
| wdSortFieldKoreaKS      | 6   | 朝鲜语 KS 顺序。 |
| wdSortFieldNumeric      | 1   | 数字顺序。       |
| wdSortFieldStroke       | 5   | 笔划顺序。       |
| wdSortFieldSyllable     | 3   | 音节顺序。       |
