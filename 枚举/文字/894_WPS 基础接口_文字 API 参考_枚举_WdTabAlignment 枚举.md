# WdTabAlignment 枚举

指定要应用的制表位对齐方式。

| 名称              | 值  | 说明         |
|-------------------|-----|--------------|
| wdAlignTabBar     | 4   | 竖线对齐。   |
| wdAlignTabCenter  | 1   | 居中对齐。   |
| wdAlignTabDecimal | 3   | 小数点对齐。 |
| wdAlignTabLeft    | 0   | 左对齐。     |
| wdAlignTabList    | 6   | 列表对齐。   |
| wdAlignTabRight   | 2   | 右对齐。     |
