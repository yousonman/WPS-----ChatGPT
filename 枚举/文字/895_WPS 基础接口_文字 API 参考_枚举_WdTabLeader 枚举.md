# WdTabLeader 枚举

指定要用作带格式制表符前导符的字符。

| 名称                 | 值  | 说明           |
|----------------------|-----|----------------|
| wdTabLeaderDashes    | 2   | 划线。         |
| wdTabLeaderDots      | 1   | 点。           |
| wdTabLeaderHeavy     | 4   | 粗线。         |
| wdTabLeaderLines     | 3   | 双线。         |
| wdTabLeaderMiddleDot | 5   | 中间点。       |
| wdTabLeaderSpaces    | 0   | 空格。默认值。 |
