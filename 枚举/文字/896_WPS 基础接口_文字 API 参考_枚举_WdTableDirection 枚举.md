# WdTableDirection 枚举

指定 WPS 对指定表格或行中的单元格进行排序的方向。

| 名称                | 值  | 说明                                         |
|---------------------|-----|----------------------------------------------|
| wdTableDirectionLtr | 1   | 对所选行进行排列，使第一列位于最左边的位置。 |
| wdTableDirectionRtl | 0   | 对所选行进行排列，使第一列位于最右边的位置。 |
