# WdTemplateType 枚举

指定模板的类型。

| 名称               | 值  | 说明           |
|--------------------|-----|----------------|
| wdAttachedTemplate | 2   | 附加模板。     |
| wdGlobalTemplate   | 1   | 全局模板。     |
| wdNormalTemplate   | 0   | 标准默认模板。 |
