# WdTextFormFieldType 枚举

指定文字型窗体域的类型。

| 名称              | 值  | 说明               |
|-------------------|-----|--------------------|
| wdCalculationText | 5   | 计算文字型域。     |
| wdCurrentDateText | 3   | 当前日期文字型域。 |
| wdCurrentTimeText | 4   | 当前时间文字型域。 |
| wdDateText        | 2   | 日期文字型域。     |
| wdNumberText      | 1   | 数字文字型域。     |
| wdRegularText     | 0   | 常规文字型域。     |
