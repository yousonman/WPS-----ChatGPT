# WdToaFormat 枚举

指定要应用于活动文档中引文目录的格式的类型。

| 名称             | 值  | 说明       |
|------------------|-----|------------|
| wdTOAClassic     | 1   | 古典格式。 |
| wdTOADistinctive | 2   | 优雅格式。 |
| wdTOAFormal      | 3   | 正式格式。 |
| wdTOASimple      | 4   | 简单格式。 |
| wdTOATemplate    | 0   | 模板格式。 |
