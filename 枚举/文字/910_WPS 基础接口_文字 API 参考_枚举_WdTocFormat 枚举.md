# WdTocFormat 枚举

指定要应用于活动文档中目录的格式的类型。

| 名称             | 值  | 说明       |
|------------------|-----|------------|
| wdTOCClassic     | 1   | 古典格式。 |
| wdTOCDistinctive | 2   | 优雅格式。 |
| wdTOCFancy       | 3   | 流行格式。 |
| wdTOCFormal      | 5   | 正式格式。 |
| wdTOCModern      | 4   | 现代格式。 |
| wdTOCSimple      | 6   | 简单格式。 |
| wdTOCTemplate    | 0   | 模板格式。 |
