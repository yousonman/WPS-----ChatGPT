# WdTofFormat 枚举

指定要应用于活动文档中图表目录的格式的类型。

| 名称             | 值  | 说明       |
|------------------|-----|------------|
| wdTOFCentered    | 3   | 居中格式。 |
| wdTOFClassic     | 1   | 古典格式。 |
| wdTOFDistinctive | 2   | 优雅格式。 |
| wdTOFFormal      | 4   | 正式格式。 |
| wdTOFSimple      | 5   | 简单格式。 |
| wdTOFTemplate    | 0   | 模板格式。 |
