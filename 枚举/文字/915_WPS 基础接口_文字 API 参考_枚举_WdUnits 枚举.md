# WdUnits 枚举

指定要使用的度量单位。

| 名称                  | 值  | 说明       |
|-----------------------|-----|------------|
| wdCell                | 12  | 单元格。   |
| wdCharacter           | 1   | 字符。     |
| wdCharacterFormatting | 13  | 字符格式。 |
| wdColumn              | 9   | 列。       |
| wdItem                | 16  | 所选项。   |
| wdLine                | 5   | 一个线段。 |
| wdParagraph           | 4   | 段落。     |
| wdParagraphFormatting | 14  | 段落格式。 |
| wdRow                 | 10  | 行。       |
| wdScreen              | 7   | 屏幕尺寸。 |
| wdSection             | 8   | 一节。     |
| wdSentence            | 3   | 句子。     |
| wdStory               | 6   | 部分。     |
| wdTable               | 15  | 一个表格。 |
| wdWindow              | 11  | 窗口。     |
| wdWord                | 2   | 字。       |
