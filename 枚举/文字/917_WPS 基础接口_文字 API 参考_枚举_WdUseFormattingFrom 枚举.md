# WdUseFormattingFrom 枚举

指定格式复制操作的源。

| 名称                     | 值  | 说明                       |
|--------------------------|-----|----------------------------|
| wdFormattingFromCurrent  | 0   | 从当前项复制源格式。       |
| wdFormattingFromPrompt   | 2   | 提示用户指定要使用的格式。 |
| wdFormattingFromSelected | 1   | 从当前选定内容复制源格式。 |
