# WdVerticalAlignment 枚举

指定要应用的垂直对齐方式的类型。

| 名称                   | 值  | 说明           |
|------------------------|-----|----------------|
| wdAlignVerticalBottom  | 3   | 靠下垂直对齐。 |
| wdAlignVerticalCenter  | 1   | 居中垂直对齐。 |
| wdAlignVerticalJustify | 2   | 两端垂直对齐。 |
| wdAlignVerticalTop     | 0   | 靠上垂直对齐。 |
