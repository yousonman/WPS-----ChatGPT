# WdVisualSelection 枚举

指定选定内容的行为，该行为基于从右向左语言文档中可视的光标移动。

| 名称                        | 值  | 说明                   |
|-----------------------------|-----|------------------------|
| wdVisualSelectionBlock      | 0   | 所有选定行的宽度相同。 |
| wdVisualSelectionContinuous | 1   | 选定内容进行换行。     |
