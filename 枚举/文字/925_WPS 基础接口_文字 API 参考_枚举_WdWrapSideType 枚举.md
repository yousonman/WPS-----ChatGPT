# WdWrapSideType 枚举

指定文档文字应该在所指定形状的两侧环绕，只在左侧或右侧环绕，还是在形状距离页边距最远的一侧环绕。

| 名称          | 值  | 说明                       |
|---------------|-----|----------------------------|
| wdWrapBoth    | 0   | 所指定形状的两侧。         |
| wdWrapLargest | 3   | 形状距离页边距最远的一侧。 |
| wdWrapLeft    | 1   | 仅限于形状的左侧。         |
| wdWrapRight   | 2   | 仅限于形状的右侧。         |
