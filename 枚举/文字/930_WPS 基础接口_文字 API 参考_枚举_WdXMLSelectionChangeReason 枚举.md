# WdXMLSelectionChangeReason 枚举

指定更改 XML 选定内容的原因。

| 名称                             | 值  | 说明                       |
|----------------------------------|-----|----------------------------|
| wdXMLSelectionChangeReasonDelete | 2   | 删除了节点。               |
| wdXMLSelectionChangeReasonInsert | 1   | 插入了新节点。             |
| wdXMLSelectionChangeReasonMove   | 0   | 移动了光标以选择不同节点。 |
