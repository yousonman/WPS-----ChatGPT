# WdXMLValidationStatus 枚举

指定按照附加的架构，元素或属性是否有效。

| 名称                        | 值          | 说明                                                                                  |
|-----------------------------|-------------|---------------------------------------------------------------------------------------|
| wdXMLValidationStatusCustom | -1072898048 | 指示 SetValidationError 方法被用于将 ValidationErrorText 属性设置为自定义文本字符串。 |
| wdXMLValidationStatusOK     | 0           | 指示按照附加的架构，XML 元素或属性有效。                                              |
