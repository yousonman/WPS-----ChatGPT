# WpsProtectionType 枚举

指定排序方式。

| 名称                   | 值  | 说明         |
|------------------------|-----|--------------|
| wpsNoProtection        | -1  | 通过名称排序 |
| wpsAllowOnlyRevisions  | 0   | 仅允许修订   |
| wpsAllowOnlyComments   | 1   | 只允许注释   |
| wpsAllowOnlyFormFields | 2   | 只允许窗体域 |
| wpsAllowOnlyReading    | 3   | 只允许阅读   |
