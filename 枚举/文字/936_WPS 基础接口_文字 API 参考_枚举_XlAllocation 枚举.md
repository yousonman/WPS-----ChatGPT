# XlAllocation 枚举

指定在对基于 OLAP 数据源的数据透视表执行模拟分析时，何时计算更改。

| 名称                  | 值  | 说明                         |
|-----------------------|-----|------------------------------|
| xlAutomaticAllocation | 2   | 在每个值更改后自动计算更改。 |
| xlManualAllocation    | 1   | 手动计算更改。               |
