# XlAllocationValue 枚举

指定在对基于 OLAP 数据源的数据透视表执行模拟分析时，要分配什么值。

| 名称                | 值  | 说明                     |
|---------------------|-----|--------------------------|
| xlAllocateIncrement | 2   | 在旧值的基础上递增。     |
| xlAllocateValue     | 1   | 输入的值除以分配的次数。 |
