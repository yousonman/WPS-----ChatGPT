# XlArrowHeadWidth 枚举

|------------------------|-------|----------------|
| xlArrowHeadWidthMedium | -4138 | 中等宽度箭头。 |
| xlArrowHeadWidthNarrow | 1     | 最窄箭头。     |
| xlArrowHeadWidthWide   | 3     | 最宽箭头。     |
