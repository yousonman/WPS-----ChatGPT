# XlCategoryType 枚举

指定分类轴的类型。

| 名称             | 值    | 说明                       |
|------------------|-------|----------------------------|
| xlAutomaticScale | -4105 | WPS 控制轴类型。           |
| xlCategoryScale  | 2     | 轴按一组任意类别组合数据。 |
| xlTimeScale      | 3     | 轴按时间刻度组合数据。     |
