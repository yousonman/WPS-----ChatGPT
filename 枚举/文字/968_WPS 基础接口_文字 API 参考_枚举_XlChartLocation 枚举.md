# XlChartLocation 枚举

指定在何处重定位图表。

| 名称                 | 值  | 说明                       |
|----------------------|-----|----------------------------|
| xlLocationAsNewSheet | 1   | 将图表移动到新工作表。     |
| xlLocationAsObject   | 2   | 将图表嵌入到现有工作表中。 |
| xlLocationAutomatic  | 3   | ET 控制图表位置。          |
