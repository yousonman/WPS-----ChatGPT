# XlChartPictureType 枚举

指定在柱形、条形图或图例项标示上如何显示图片。

| 名称         | 值  | 说明                                                     |
|--------------|-----|----------------------------------------------------------|
| xlStack      | 2   | 调整图片大小，以在最长的堆积条形图中最多重复显示 15 次。 |
| xlStackScale | 3   | 将图片大小调整为指定的单位数，并在条形图长度内重复显示。 |
| xlStretch    | 1   | 将图片拉伸至堆积条形图的全长。                           |
