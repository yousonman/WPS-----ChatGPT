# XlColorIndex 枚举

指定所选功能（如边框、字体或填充）的颜色。

| 名称                  | 值    | 说明       |
|-----------------------|-------|------------|
| xlColorIndexAutomatic | -4105 | 自动配色。 |
| xlColorIndexNone      | -4142 | 无颜色。   |
