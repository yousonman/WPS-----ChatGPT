# XlCommandUnderlines 枚举

指定 ET for the Macintosh 中命令加下划线的状态。

| 名称                         | 值    | 说明                        |
|------------------------------|-------|-----------------------------|
| xlCommandUnderlinesAutomatic | -4105 | ET 控制命令加下划线的显示。 |
| xlCommandUnderlinesOff       | -4146 | 不显示命令加下划线。        |
| xlCommandUnderlinesOn        | 1     | 显示命令加下划线。          |
