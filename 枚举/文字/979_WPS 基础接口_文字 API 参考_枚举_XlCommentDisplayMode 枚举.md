# XlCommentDisplayMode 枚举

指定单元格显示批注和批注标识符的方式。

| 名称                   | 值  | 说明                                             |
|------------------------|-----|--------------------------------------------------|
| xlCommentAndIndicator  | 1   | 任何时候都显示批注和标识符。                     |
| xlCommentIndicatorOnly | -1  | 只显示标识符。鼠标指针在单元格上移动时显示批注。 |
| xlNoIndicator          | 0   | 任何时候都不显示批注也不显示标识符。             |
