# XlContainsOperator 枚举

指定函数使用的运算符。

| 名称             | 值  | 说明             |
|------------------|-----|------------------|
| xlBeginsWith     | 2   | 以指定的值开始。 |
| xlContains       | 0   | 包含指定的值。   |
| xlDoesNotContain | 1   | 不包含指定的值。 |
| xlEndsWith       | 3   | 以指定的值结束   |
