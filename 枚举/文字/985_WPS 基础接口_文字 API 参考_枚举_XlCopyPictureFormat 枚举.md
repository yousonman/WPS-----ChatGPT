# XlCopyPictureFormat 枚举

指定复制的图片的格式。

| 名称      | 值    | 说明                           |
|-----------|-------|--------------------------------|
| xlBitmap  | 2     | 位图（.bmp、.jpg、.gif）。     |
| xlPicture | -4147 | 绘制图片（.png、.wmf、.mix）。 |
