# MsoBarType 枚举

指定命令栏的类型。

| 名称              | 值  | 说明         |
|-------------------|-----|--------------|
| msoBarTypeMenuBar | 1   | 菜单栏。     |
| msoBarTypeNormal  | 0   | 默认菜单栏。 |
| msoBarTypePopup   | 2   | 快捷菜单。   |
