# MsoClipboardFormat 枚举

指定剪贴板格式。

| 名称                        | 值  | 说明                           |
|-----------------------------|-----|--------------------------------|
| msoClipboardFormatHTML      | 2   | HTML 格式。                    |
| msoClipboardFormatMixed     | -2  | 只返回值，表示其他状态的组合。 |
| msoClipboardFormatNative    | 1   | 本机格式。                     |
| msoClipboardFormatPlainText | 4   | 纯文本格式。                   |
| msoClipboardFormatRTF       | 3   | RTF 格式。                     |
