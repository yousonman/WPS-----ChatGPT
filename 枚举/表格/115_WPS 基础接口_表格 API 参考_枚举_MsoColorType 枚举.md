# MsoColorType 枚举

指定颜色类型。

| 名称               | 值  | 说明                                     |
|--------------------|-----|------------------------------------------|
| msoColorTypeCMS    | 4   | “颜色管理系统”颜色类型。                 |
| msoColorTypeCMYK   | 3   | 颜色由蓝绿色、洋红、黄色和黑色的值确定。 |
| msoColorTypeInk    | 5   | 不支持。                                 |
| msoColorTypeMixed  | -2  | 不支持。                                 |
| msoColorTypeRGB    | 1   | 颜色由红色、绿色和蓝色的值确定。         |
| msoColorTypeScheme | 2   | 颜色由应用程序特定的方案定义。           |
