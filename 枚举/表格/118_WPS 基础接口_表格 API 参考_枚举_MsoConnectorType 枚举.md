# MsoConnectorType 枚举

指定连接符的类型。

| 名称                  | 值  | 说明                           |
|-----------------------|-----|--------------------------------|
| msoConnectorCurve     | 3   | 曲线连接符。                   |
| msoConnectorElbow     | 2   | 肘形连接符。                   |
| msoConnectorStraight  | 1   | 直线连接符。                   |
| msoConnectorTypeMixed | -2  | 只返回值，表示其他状态的组合。 |
