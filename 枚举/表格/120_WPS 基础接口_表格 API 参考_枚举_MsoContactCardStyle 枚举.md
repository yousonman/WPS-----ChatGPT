# MsoContactCardStyle 枚举

指定如何显示联系人卡片。

| 名称                | 值  | 说明                       |
|---------------------|-----|----------------------------|
| msoContactCardFull  | 1   | 联系人卡片显示为完整卡片。 |
| msoContactCardHover | 0   | 联系人卡片显示为悬浮卡片。 |
