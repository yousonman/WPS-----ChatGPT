# MsoDocInspectorStatus 枚举

代表运行文档检查器模块的结果。

| 名称                            | 值  | 说明                                               |
|---------------------------------|-----|----------------------------------------------------|
| msoDocInspectorStatusDocOk      | 0   | 指示文档检查器模块未返回问题或错误。               |
| msoDocInspectorStatusError      | 2   | 指示文档检查器模块返回了错误。                     |
| msoDocInspectorStatusIssueFound | 1   | 指示文档检查器模块找到了一个或多个搜索条件匹配项。 |
