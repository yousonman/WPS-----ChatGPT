# MsoFileDialogType 枚举

指定 **FileDialog** 对象的类型。

| 名称                      | 值  | 说明                    |
|---------------------------|-----|-------------------------|
| msoFileDialogFilePicker   | 3   | “文件选取器” 对话框。   |
| msoFileDialogFolderPicker | 4   | “文件夹选取器” 对话框。 |
| msoFileDialogOpen         | 1   | “打开” 对话框。         |
| msoFileDialogSaveAs       | 2   | “另存为” 对话框。       |
