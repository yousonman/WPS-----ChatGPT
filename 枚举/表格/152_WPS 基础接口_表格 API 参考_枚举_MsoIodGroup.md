# MsoIodGroup

指定用于定义 IOD（按需安装）组的常量。

| 名称                  | 值  | 说明              |
|-----------------------|-----|-------------------|
| msoIodGroupPIAs       | 0   | PIA 组。          |
| msoIodGroupVSTOR35Mgd | 1   | VSTO 3.5 托管组。 |
| msoIodGroupVSTOR40Mgd | 2   | VSTO 4.0 托管组。 |
