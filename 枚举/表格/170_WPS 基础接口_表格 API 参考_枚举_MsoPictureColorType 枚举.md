# MsoPictureColorType 枚举

指定应用于图片的颜色转换设置。

| 名称                    | 值  | 说明           |
|-------------------------|-----|----------------|
| msoPictureAutomatic     | 1   | 默认颜色转换。 |
| msoPictureBlackAndWhite | 3   | 黑白转换。     |
| msoPictureGrayscale     | 2   | 灰度转换。     |
| msoPictureMixed         | -2  | 混和转换。     |
| msoPictureWatermark     | 4   | 水印转换。     |
