# MsoSegmentType 枚举

指定线段的类型。与 **FreeformBuilder** 对象的 **Insert** 和 **AddNodes** 方法一起使用。

| 名称            | 值  | 说明   |
|-----------------|-----|--------|
| msoSegmentCurve | 1   | 曲线。 |
| msoSegmentLine  | 0   | 直线。 |
