# MsoShadowStyle 枚举

指定阴影效果的类型。

| 名称                      | 值  | 说明                                   |
|---------------------------|-----|----------------------------------------|
| msoShadowStyleInnerShadow | 1   | 指定内部阴影效果。                     |
| msoShadowStyleMixed       | -2  | 指定内部阴影效果和外部阴影效果的组合。 |
| msoShadowStyleOuterShadow | 2   | 指定外部阴影效果。                     |
