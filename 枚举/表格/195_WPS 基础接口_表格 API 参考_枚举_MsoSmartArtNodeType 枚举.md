# MsoSmartArtNodeType 枚举

指定用于定义 SmartArt 节点类型的常量。

| 名称                         | 值  | 说明                                   |
|------------------------------|-----|----------------------------------------|
| msoSmartArtNodeTypeAssistant | 2   | 定义一个可在分层图表中使用的辅助节点。 |
| msoSmartArtNodeTypeDefault   | 1   | 默认节点类型。                         |
