# MsoTextCaps 枚举

指定文本的大写设置。

| 名称         | 值  | 说明                                       |
|--------------|-----|--------------------------------------------|
| msoAllCaps   | 2   | 将文本以全部大写字母的形式显示。           |
| msoCapsMixed | -2  | 将文本以大写字母和小写字母混合的形式显示。 |
| msoNoCaps    | 0   | 将文本以不带大写字母的形式显示。           |
| msoSmallCaps | 1   | 将文本以全部小写字母的形式显示。           |
