# MsoTextCharWrap 枚举

指示文字环绕的类型。

| 名称                | 值  | 说明                                     |
|---------------------|-----|------------------------------------------|
| msoCharWrapMixed    | -2  | 指定混和的文字环绕。                     |
| msoCustomCharWrap   | 3   | 指定自定义文字环绕方案。                 |
| msoNoCharWrap       | 0   | 指定无文字环绕。                         |
| msoStandardCharWrap | 1   | 指定在对象的标准边界周围环绕文字。       |
| msoStrictCharWrap   | 2   | 指定文字环绕符合中文和日文等语言的限制。 |
