# MsoZOrderCmd 枚举

指定要将形状移动到的相对于其他形状的 z-顺序位置。

| 名称                  | 值  | 说明                 |
|-----------------------|-----|----------------------|
| msoBringForward       | 2   | 使形状上移一层。     |
| msoBringInFrontOfText | 4   | 使形状浮于文字上方。 |
| msoBringToFront       | 0   | 使形状置于顶层。     |
| msoSendBackward       | 3   | 使形状下移一层。     |
| msoSendBehindText     | 5   | 使形状衬于文字下方。 |
| msoSendToBack         | 1   | 使形状置于底层。     |
