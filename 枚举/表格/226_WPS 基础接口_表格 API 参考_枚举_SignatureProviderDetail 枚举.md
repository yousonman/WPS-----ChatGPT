# SignatureProviderDetail 枚举

指定签名提供程序的属性。

| 名称                    | 值  | 说明                                       |
|-------------------------|-----|--------------------------------------------|
| sigprovdetHashAlgorithm | 1   | 用于对文件中的数据进行散列运算的哈希算法。 |
| sigprovdetUIOnly        | 2   | 指示签名提供程序只使用自定义用户界面。     |
| sigprovdetUrl           | 0   | 签名提供程序的 URL。                       |
