# XlAboveBelow 枚举

指定值是高于还是低于平均值。

| 名称                | 值  | 说明               |
|---------------------|-----|--------------------|
| XlAboveAverage      | 0   | 高于平均值。       |
| XlAboveStdDev       | 1   | 高于标准偏差。     |
| XlBelowAverage      | 0   | 低于平均值。       |
| XlBelowStdDev       | 1   | 低于标准偏差。     |
| XlEqualAboveAverage | 0   | 等于或高于平均值。 |
| XlEqualBelowAverage | 0   | 等于或低于平均值。 |
