# XlAllocationMethod 枚举

指定在对基于 OLAP 数据源的数据透视表执行模拟分析时，要用来分配值的方法。

| 名称                 | 值  | 说明           |
|----------------------|-----|----------------|
| xlEqualAllocation    | 1   | 使用平均分配。 |
| xlWeightedAllocation | 2   | 使用加权分配。 |
