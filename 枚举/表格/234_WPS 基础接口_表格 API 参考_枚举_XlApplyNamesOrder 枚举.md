# XlApplyNamesOrder 枚举

指定用行方向区域名称和列方向区域名称取代单元格引用时，首先列出哪个区域名称。

| 名称            | 值  | 说明             |
|-----------------|-----|------------------|
| xlColumnThenRow | 2   | 列在行之前列出。 |
| xlRowThenColumn | 1   | 行在列之前列出。 |
