# XlArrangeStyle 枚举

指定窗口在屏幕上的排列方式。

| 名称                     | 值    | 说明               |
|--------------------------|-------|--------------------|
| xlArrangeStyleCascade    | 7     | 层叠窗口。         |
| xlArrangeStyleHorizontal | -4128 | 水平排列窗口。     |
| xlArrangeStyleTiled      | 1     | 默认值。平铺窗口。 |
| xlArrangeStyleVertical   | -4166 | 垂直排列窗口。     |
