# XlAxisType 枚举

指定坐标轴类型。

| 名称         | 值  | 说明                 |
|--------------|-----|----------------------|
| xlCategory   | 1   | 坐标轴显示类别。     |
| xlSeriesAxis | 3   | 坐标轴显示数据系列。 |
| xlValue      | 2   | 坐标轴显示值。       |
