# XlBackground 枚举

指定图表文本的背景类型。

| 名称                    | 值    | 说明          |
|-------------------------|-------|---------------|
| xlBackgroundAutomatic   | -4105 | ET 控制背景。 |
| xlBackgroundOpaque      | 3     | 不透明背景。  |
| xlBackgroundTransparent | 2     | 透明背景。    |
