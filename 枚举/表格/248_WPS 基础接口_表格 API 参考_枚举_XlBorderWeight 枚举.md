# XlBorderWeight 枚举

指定某一区域周围的边框的粗细。

| 名称       | 值    | 说明                 |
|------------|-------|----------------------|
| xlHairline | 1     | 细线（最细的边框）。 |
| xlMedium   | -4138 | 中等。               |
| xlThick    | 4     | 粗（最宽的边框）。   |
| xlThin     | 2     | 细。                 |
