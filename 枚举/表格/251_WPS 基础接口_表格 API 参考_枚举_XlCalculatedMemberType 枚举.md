# XlCalculatedMemberType 枚举

指定数据透视表中计算成员的类型。

| 名称               | 值  | 说明                                      |
|--------------------|-----|-------------------------------------------|
| xlCalculatedMember | 0   | 成员使用多维表达式 (MDX) 公式。           |
| xlCalculatedSet    | 1   | 成员在多维数据集字段中包含集的 MDX 公式。 |
