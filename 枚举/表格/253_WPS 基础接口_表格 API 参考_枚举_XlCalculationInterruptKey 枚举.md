# XlCalculationInterruptKey 枚举

指定中断重新计算的键。

| 名称     | 值  | 说明                         |
|----------|-----|------------------------------|
| xlAnyKey | 2   | 按任意键中断重新计算。       |
| xlEscKey | 1   | 按 Esc 键中断重新计算。      |
| xlNoKey  | 0   | 按任何键都不能中断重新计算。 |
