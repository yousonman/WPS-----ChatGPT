# XlCalculationState 枚举

指定应用程序的计算状态。

| 名称          | 值  | 说明                                         |
|---------------|-----|----------------------------------------------|
| xlCalculating | 1   | 正在计算。                                   |
| xlDone        | 0   | 计算完成。                                   |
| xlPending     | 2   | 已进行会触发计算的更改，但还未执行重新计算。 |
