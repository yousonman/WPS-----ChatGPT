# XlChartPicturePlacement 枚举

指定用户所选图片在三维条形图或柱形图中的某个条形上的位置。

| 名称         | 值  | 说明                 |
|--------------|-----|----------------------|
| xlAllFaces   | 7   | 在所有表面上显示。   |
| xlEnd        | 2   | 在末端显示。         |
| xlEndSides   | 3   | 在末端和侧面上显示。 |
| xlFront      | 4   | 在前端显示。         |
| xlFrontEnd   | 6   | 在前端和末端显示。   |
| xlFrontSides | 5   | 在前端和侧面上显示。 |
| xlSides      | 1   | 在侧面上显示。       |
