# XlChartSplitType 枚举

指定在饼图的第二个图表或饼图的条形中显示的值。

| 名称                  | 值  | 说明                                                                       |
|-----------------------|-----|----------------------------------------------------------------------------|
| xlSplitByCustomSplit  | 4   | 在第二个图表中显示任意幻灯片。                                             |
| xlSplitByPercentValue | 3   | 第二个图表显示小于总值的某个百分比的值。该百分比由 SplitValue 属性指定。   |
| xlSplitByPosition     | 1   | 第二个图表显示数据系列中的最小值。要显示的值的个数由 SplitValue 属性指定。 |
| xlSplitByValue        | 2   | 第二个图表显示的值小于 SplitValue 属性指定的值。                           |
