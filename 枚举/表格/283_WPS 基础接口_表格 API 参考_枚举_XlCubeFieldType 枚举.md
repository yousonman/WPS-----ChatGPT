# XlCubeFieldType 枚举

指定 OLAP 字段是层次结构、集合还是度量字段。

| 名称        | 值  | 说明                  |
|-------------|-----|-----------------------|
| xlHierarchy | 1   | OLAP 字段是层次结构。 |
| xlMeasure   | 2   | OLAP 字段是度量。     |
| xlSet       | 3   | OLAP 字段是集合。     |
