# XlDataBarFillType 枚举

指定如何对数据条填充颜色。

| 名称                  | 值  | 说明                 |
|-----------------------|-----|----------------------|
| xlDataBarFillGradient | 1   | 对数据条填充渐变色。 |
| xlDataBarFillSolid    | 0   | 对数据条填充纯色。   |
