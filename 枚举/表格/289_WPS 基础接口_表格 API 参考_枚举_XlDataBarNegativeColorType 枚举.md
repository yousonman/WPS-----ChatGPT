# XlDataBarNegativeColorType 枚举

指定是否使用与正数据条相同的边框和填充色。

| 名称                    | 值  | 说明                                                                                                                          |
|-------------------------|-----|-------------------------------------------------------------------------------------------------------------------------------|
| xlDataBarColor          | 0   | 使用 “负值和坐标轴设置” 对话框中指定的颜色，或者使用由 NegativeBarFormat 对象的 ColorType 和 BorderColorType 属性指定的颜色。 |
| xlDataBarSameAsPositive | 1   | 使用与正数据条相同的颜色。                                                                                                    |
