# XlDataSeriesDate 枚举

指定要应用于数据系列的日期的类型。

| 名称      | 值  | 说明   |
|-----------|-----|--------|
| xlDay     | 1   | 日     |
| xlMonth   | 3   | 月     |
| xlWeekday | 2   | 工作日 |
| xlYear    | 4   | 年     |
