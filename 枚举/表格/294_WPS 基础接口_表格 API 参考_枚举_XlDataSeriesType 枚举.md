# XlDataSeriesType 枚举

指定要创建的数据系列。

| 名称               | 值    | 说明                                                         |
|--------------------|-------|--------------------------------------------------------------|
| xlAutoFill         | 4     | 按照“自动填充”设置对系列进行填充。                           |
| xlChronological    | 3     | 用数据值进行填充。                                           |
| xlDataSeriesLinear | -4132 | 扩展值，假定一个加法级数（例如，“1, 2”被扩展为“3, 4, 5”）。  |
| xlGrowth           | 2     | 扩展值，假定一个乘法级数（例如，“1, 2”被扩展为“4, 8, 16”）。 |
