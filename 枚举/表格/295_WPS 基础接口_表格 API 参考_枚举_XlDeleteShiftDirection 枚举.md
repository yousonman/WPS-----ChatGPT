# XlDeleteShiftDirection 枚举

指定如何移动单元格来替换删除的单元格。

| 名称          | 值    | 说明             |
|---------------|-------|------------------|
| xlShiftToLeft | -4159 | 单元格向左移动。 |
| xlShiftUp     | -4162 | 单元格向上移动。 |
