# XlDupeUnique 枚举

指定应显示重复值还是唯一值。

| 名称        | 值  | 说明         |
|-------------|-----|--------------|
| xlDuplicate | 1   | 显示重复值。 |
| xlUnique    | 0   | 显示唯一值。 |
