# XlEnableSelection 枚举

指定可在工作表中选择的内容。

| 名称             | 值    | 说明                   |
|------------------|-------|------------------------|
| xlNoRestrictions | 0     | 可以选择任何内容。     |
| xlNoSelection    | -4142 | 不能选择任何内容。     |
| xlUnlockedCells  | 1     | 只能选择未锁定单元格。 |
