# XlErrorBarDirection 枚举

指定接收误差线的轴值。

| 名称 | 值    | 说明                               |
|------|-------|------------------------------------|
| xlX  | -4168 | 误差线平行于 Y 轴，长度为 X 轴值。 |
| xlY  | 1     | 误差线平行于 X 轴，长度为 Y 轴值。 |
