# XlErrorBarInclude 枚举

指定要包含的误差线部分。

| 名称                         | 值    | 说明           |
|------------------------------|-------|----------------|
| xlErrorBarIncludeBoth        | 1     | 正负误差区域。 |
| xlErrorBarIncludeMinusValues | 3     | 仅负误差区域。 |
| xlErrorBarIncludeNone        | -4142 | 无误差线区域。 |
| xlErrorBarIncludePlusValues  | 2     | 仅正误差区域。 |
