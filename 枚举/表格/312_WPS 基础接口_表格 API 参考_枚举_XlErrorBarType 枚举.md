# XlErrorBarType 枚举

指定误差线标记的区域。

| 名称                     | 值    | 说明                           |
|--------------------------|-------|--------------------------------|
| xlErrorBarTypeCustom     | -4114 | 区域由固定值或单元格值设置。   |
| xlErrorBarTypeFixedValue | 1     | 固定长度误差线。               |
| xlErrorBarTypePercent    | 2     | 误差线覆盖的区域百分比。       |
| xlErrorBarTypeStDev      | -4155 | 显示指定数目的标准偏差的区域。 |
| xlErrorBarTypeStError    | 4     | 显示标准误差区域。             |
