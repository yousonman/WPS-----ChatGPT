# XlFormatFilterTypes 枚举

指定格式筛选的类型。

| 名称                | 值  | 说明           |
|---------------------|-----|----------------|
| FilterBottom        | 0   | 下筛选。       |
| FilterBottomPercent | 2   | 下百分比筛选。 |
| FilterTop           | 1   | 上筛选。       |
| FilterTopPercent    | 3   | 上百分比筛选。 |
