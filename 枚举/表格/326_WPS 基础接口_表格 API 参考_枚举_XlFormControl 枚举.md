# XlFormControl 枚举

指定表单控件的类型。

| 名称            | 值  | 说明       |
|-----------------|-----|------------|
| xlButtonControl | 0   | 按钮。     |
| xlCheckBox      | 1   | 复选框。   |
| xlDropDown      | 2   | 组合框。   |
| xlEditBox       | 3   | 文本框。   |
| xlGroupBox      | 4   | 分组框。   |
| xlLabel         | 5   | 标签。     |
| xlListBox       | 6   | 列表框。   |
| xlOptionButton  | 7   | 选项按钮。 |
| xlScrollBar     | 8   | 滚动条。   |
| xlSpinner       | 9   | 微调按钮。 |
