# XlGradientFillType 枚举

指定 **gradient fill** 的类型。

| 名称               | 值  | 说明                         |
|--------------------|-----|------------------------------|
| GradientFillLinear | 0   | 渐变以直线填充。             |
| GradientFillPath   | 1   | 渐变以非线性或曲线路径填充。 |
