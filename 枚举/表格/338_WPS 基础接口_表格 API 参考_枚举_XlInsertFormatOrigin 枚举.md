# XlInsertFormatOrigin 枚举

指定从何处复制插入行的格式。

| 名称                     | 值  | 说明                            |
|--------------------------|-----|---------------------------------|
| xlFormatFromLeftOrAbove  | 0   | 从上方和/或左侧单元格复制格式。 |
| xlFormatFromRightOrBelow | 1   | 从下方和/或右侧单元格复制格式。 |
