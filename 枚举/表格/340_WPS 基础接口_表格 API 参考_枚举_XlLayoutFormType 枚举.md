# XlLayoutFormType 枚举

为指定的数据透视表项指定显示方式，即以表格式还是以大纲格式显示。

| 名称      | 值  | 说明                                                              |
|-----------|-----|-------------------------------------------------------------------|
| xlOutline | 1   | LayoutSubtotalLocation 属性指定分类汇总在数据透视表中出现的位置。 |
| xlTabular | 0   | 默认值。                                                          |
