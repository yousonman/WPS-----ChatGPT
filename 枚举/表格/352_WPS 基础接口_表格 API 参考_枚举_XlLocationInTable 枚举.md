# XlLocationInTable 枚举

指定数据透视表中包含区域左上角的部分。

| 名称           | 值    | 说明     |
|----------------|-------|----------|
| xlColumnHeader | -4110 | 列标题   |
| xlColumnItem   | 5     | 列数据项 |
| xlDataHeader   | 3     | 数据标题 |
| xlDataItem     | 7     | 数据项   |
| xlPageHeader   | 2     | 页面页眉 |
| xlPageItem     | 6     | 页面项   |
| xlRowHeader    | -4153 | 行标题   |
| xlRowItem      | 4     | 行数据项 |
| xlTableBody    | 8     | 表正文   |
