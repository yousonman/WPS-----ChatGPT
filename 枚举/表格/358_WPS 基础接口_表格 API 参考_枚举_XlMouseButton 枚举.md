# XlMouseButton 枚举

指定按下了哪个鼠标按钮。

| 名称              | 值  | 说明                             |
|-------------------|-----|----------------------------------|
| xlNoButton        | 0   | 没有按任何按钮。                 |
| xlPrimaryButton   | 1   | 按下主按钮（通常为鼠标左按钮）。 |
| xlSecondaryButton | 2   | 按下辅按钮（通常为鼠标右按钮）。 |
