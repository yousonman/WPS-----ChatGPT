# XlOartHorizontalOverflow 枚举

指定文本框的水平溢出设置。

| 名称                             | 值  | 说明                           |
|----------------------------------|-----|--------------------------------|
| xlOartHorizontalOverflowClip     | 1   | 隐藏水平方向溢出文本框的文本。 |
| xlOartHorizontalOverflowOverflow | 0   | 允许文本在水平方向溢出文本框。 |
