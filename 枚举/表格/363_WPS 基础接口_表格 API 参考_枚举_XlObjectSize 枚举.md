# XlObjectSize 枚举

指定图表为适应页面大小而进行缩放的方式。

| 名称         | 值  | 说明                                                               |
|--------------|-----|--------------------------------------------------------------------|
| xlFitToPage  | 2   | 尽可能大地打印图表，并保持如屏幕所示的该图表的高度对宽度的比例。   |
| xlFullPage   | 3   | 按照与页面相适应的大小打印图表，并根据需要调整其高度对宽度的比例。 |
| xlScreenSize | 1   | 以屏幕显示大小打印图表。                                           |
