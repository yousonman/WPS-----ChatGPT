# XlOLEType 枚举

指定 OLE 对象类型。

| 名称         | 值  | 说明            |
|--------------|-----|-----------------|
| xlOLEControl | 2   | ActiveX 控件    |
| xlOLEEmbed   | 1   | 嵌入式 OLE 对象 |
| xlOLELink    | 0   | 链接 OLE 对象   |
