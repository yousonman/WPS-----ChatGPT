# XlParameterType 枚举

为指定的查询表指定确定参数值的方式。

| 名称       | 值  | 说明                                                                |
|------------|-----|---------------------------------------------------------------------|
| xlConstant | 1   | 使用 Value 参数指定的值。                                           |
| xlPrompt   | 0   | 显示提示用户输入值的对话框。 Value 参数指定的是对话框中显示的文字。 |
| xlRange    | 2   | 使用区域左上角单元格的值。 Value 参数指定的是一个 Range 对象。      |
