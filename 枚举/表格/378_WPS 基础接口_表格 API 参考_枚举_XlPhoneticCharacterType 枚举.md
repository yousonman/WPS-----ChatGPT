# XlPhoneticCharacterType 枚举

指定单元格中拼音文字的类型。

| 名称           | 值  | 说明         |
|----------------|-----|--------------|
| xlHiragana     | 2   | 平假名       |
| xlKatakana     | 1   | 片假名       |
| xlKatakanaHalf | 0   | 半尺寸片假名 |
| xlNoConversion | 3   | 无转换       |
