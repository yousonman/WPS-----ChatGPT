# XlPictureAppearance 枚举

指定图片的复制方式。

| 名称      | 值  | 说明                             |
|-----------|-----|----------------------------------|
| xlPrinter | 2   | 图片按其打印效果进行复制。       |
| xlScreen  | 1   | 图片尽可能按其屏幕显示进行复制。 |
