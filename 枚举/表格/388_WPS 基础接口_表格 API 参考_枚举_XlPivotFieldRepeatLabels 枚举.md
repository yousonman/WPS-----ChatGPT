# XlPivotFieldRepeatLabels 枚举

指定是否重复数据透视表中的所有字段项目标签。

| 名称                | 值  | 说明               |
|---------------------|-----|--------------------|
| xlDoNotRepeatLabels | 1   | 不重复项目标签。   |
| xlRepeatLabels      | 2   | 重复所有项目标签。 |
