# XlPivotTableSourceType 枚举

指定报告数据源。

| 名称            | 值    | 说明                               |
|-----------------|-------|------------------------------------|
| xlConsolidation | 3     | 多重合并计算数据区域。             |
| xlDatabase      | 1     | ET 列表或数据库。                  |
| xlExternal      | 2     | 其他应用程序中的数据。             |
| xlPivotTable    | -4148 | 与另一数据透视表相同来源。         |
| xlScenario      | 4     | 数据基于使用方案管理器创建的方案。 |
