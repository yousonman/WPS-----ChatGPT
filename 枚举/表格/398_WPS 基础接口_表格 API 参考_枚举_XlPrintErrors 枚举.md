# XlPrintErrors 枚举

指定显示的打印错误的类型。

| 名称                   | 值  | 说明                   |
|------------------------|-----|------------------------|
| xlPrintErrorsBlank     | 1   | 打印错误为空白。       |
| xlPrintErrorsDash      | 2   | 打印错误显示为划线。   |
| xlPrintErrorsDisplayed | 0   | 显示全部打印错误。     |
| xlPrintErrorsNA        | 3   | 打印错误显示为不可用。 |
