# XlPrintLocation 枚举

指定表中批注的打印方式。

| 名称              | 值    | 说明                           |
|-------------------|-------|--------------------------------|
| xlPrintInPlace    | 16    | 批注打印在其插入工作表的位置。 |
| xlPrintNoComments | -4142 | 不打印批注。                   |
| xlPrintSheetEnd   | 1     | 批注打印为工作表末尾的尾注。   |
