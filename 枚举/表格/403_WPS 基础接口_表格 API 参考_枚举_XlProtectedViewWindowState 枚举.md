# XlProtectedViewWindowState 枚举

指定 **“受保护的视图”** 窗口的状态。

| 名称                           | 值  | 说明   |
|--------------------------------|-----|--------|
| xlProtectedViewWindowMaximized | 2   | 最大化 |
| xlProtectedViewWindowMinimized | 1   | 最小化 |
| xlProtectedViewWindowNormal    | 0   | 正常   |
