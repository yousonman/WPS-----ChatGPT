# XlReferenceType 枚举

指定转换公式时的单元格引用样式。

| 名称              | 值  | 说明                       |
|-------------------|-----|----------------------------|
| xlAbsolute        | 1   | 转换为绝对行和列样式。     |
| xlAbsRowRelColumn | 2   | 转换为绝对行和相对列样式。 |
| xlRelative        | 4   | 转换为相对行和列样式。     |
| xlRelRowAbsColumn | 3   | 转换为相对行和绝对列样式。 |
