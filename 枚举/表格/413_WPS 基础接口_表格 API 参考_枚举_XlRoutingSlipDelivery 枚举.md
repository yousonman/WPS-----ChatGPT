# XlRoutingSlipDelivery 枚举

指定传送传递方法。

| 名称              | 值  | 说明                   |
|-------------------|-----|------------------------|
| xlAllAtOnce       | 2   | 同时传递给所有收件人。 |
| xlOneAfterAnother | 1   | 逐个传递给收件人。     |
