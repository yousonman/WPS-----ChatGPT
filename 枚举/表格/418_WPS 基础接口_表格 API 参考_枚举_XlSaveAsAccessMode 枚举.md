# XlSaveAsAccessMode 枚举

指定“另存为”函数的访问模式。

| 名称        | 值  | 说明                     |
|-------------|-----|--------------------------|
| xlExclusive | 3   | 独占模式                 |
| xlNoChange  | 1   | 默认值（不更改访问模式） |
| xlShared    | 2   | 共享列表                 |
