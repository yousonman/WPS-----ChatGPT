# XlSaveConflictResolution 枚举

指定更新共享工作簿时解决冲突的方式。

| 名称                  | 值  | 说明                         |
|-----------------------|-----|------------------------------|
| xlLocalSessionChanges | 2   | 总是接受本地用户所做的更改。 |
| xlOtherSessionChanges | 3   | 总是拒绝本地用户所做的更改。 |
| xlUserResolution      | 1   | 弹出对话框请求用户解决冲突。 |
