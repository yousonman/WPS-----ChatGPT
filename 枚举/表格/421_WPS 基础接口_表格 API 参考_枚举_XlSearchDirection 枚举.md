# XlSearchDirection 枚举

指定搜索区域时的搜索方向。

| 名称       | 值  | 说明                     |
|------------|-----|--------------------------|
| xlNext     | 1   | 在区域中搜索下一匹配值。 |
| xlPrevious | 2   | 在区域中搜索上一匹配值。 |
