# XlSizeRepresents 枚举

指定气泡图上气泡大小表示的内容。

| 名称          | 值  | 说明         |
|---------------|-----|--------------|
| xlSizeIsArea  | 1   | 气泡的面积。 |
| xlSizeIsWidth | 2   | 气泡的宽度。 |
