# XlSlicerSort 枚举

指定在切片器中显示的项是否排序，如果排序，则是按项标题升序还是降序排序。

| 名称                        | 值  | 说明                             |
|-----------------------------|-----|----------------------------------|
| xlSlicerSortAscending       | 2   | 切片器项按项标题升序排序。       |
| xlSlicerSortDataSourceOrder | 1   | 切片器项按数据源提供的顺序显示。 |
| xlSlicerSortDescending      | 3   | 切片器项按项标题降序排序。       |
