# XlSortDataOption 枚举

指定文本的排序方式。

| 名称                | 值  | 说明                                   |
|---------------------|-----|----------------------------------------|
| xlSortNormal        | 0   | 默认值。分别对数字和文本数据进行排序。 |
| xlSortTextAsNumbers | 1   | 将文本作为数字型数据进行排序。         |
