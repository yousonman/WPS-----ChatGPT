# XlSortOrder 枚举

为指定字段或范围指定排序顺序。

| 名称         | 值  | 说明                               |
|--------------|-----|------------------------------------|
| xlAscending  | 1   | 按升序对指定字段排序。这是默认值。 |
| xlDescending | 2   | 按降序对指定字段排序。             |
