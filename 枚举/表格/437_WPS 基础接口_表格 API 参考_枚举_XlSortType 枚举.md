# XlSortType 枚举

指定要排序的元素。仅在对数据透视表排序时才使用该参数。

| 名称         | 值  | 说明                     |
|--------------|-----|--------------------------|
| xlSortLabels | 2   | 按标签对数据透视表排序。 |
| xlSortValues | 1   | 按值对数据透视表排序。   |
