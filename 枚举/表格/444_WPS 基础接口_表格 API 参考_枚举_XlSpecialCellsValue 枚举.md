# XlSpecialCellsValue 枚举

指定结果中包括具有特定类型值的单元格。

| 名称         | 值  | 说明                 |
|--------------|-----|----------------------|
| xlErrors     | 16  | 有错误的单元格。     |
| xlLogical    | 4   | 具有逻辑值的单元格。 |
| xlNumbers    | 1   | 具有数值的单元格。   |
| xlTextValues | 2   | 具有文本的单元格。   |
