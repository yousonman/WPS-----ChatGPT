# XlTabPosition 枚举

指定第一个或最后一个制表位位置。

| 名称               | 值  | 说明                 |
|--------------------|-----|----------------------|
| xlTabPositionFirst | 0   | 第一个制表位位置。   |
| xlTabPositionLast  | 1   | 最后一个制表位位置。 |
