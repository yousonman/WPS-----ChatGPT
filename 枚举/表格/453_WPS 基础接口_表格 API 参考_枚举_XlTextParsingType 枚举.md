# XlTextParsingType 枚举

指定要导入查询表的文本文件中数据的列格式。

| 名称         | 值  | 说明                                     |
|--------------|-----|------------------------------------------|
| xlDelimited  | 1   | 默认值。指示文件由分隔符分隔。           |
| xlFixedWidth | 2   | 指示将文件中的数据排列在固定宽度的列中。 |
