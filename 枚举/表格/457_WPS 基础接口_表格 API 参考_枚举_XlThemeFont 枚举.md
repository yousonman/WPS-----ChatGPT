# XlThemeFont 枚举

指定要使用的主题字体。

| 名称             | 值  | 说明                 |
|------------------|-----|----------------------|
| xlThemeFontMajor | 2   | 主要。               |
| xlThemeFontMinor | 1   | 次要。               |
| xlThemeFontNone  | 0   | 不使用任何主题字体。 |
