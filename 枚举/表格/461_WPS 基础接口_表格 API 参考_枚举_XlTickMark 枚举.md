# XlTickMark 枚举

指定坐标轴的主要和次要刻度线标志的位置。

| 名称              | 值    | 说明   |
|-------------------|-------|--------|
| xlTickMarkCross   | 4     | 跨轴   |
| xlTickMarkInside  | 2     | 在轴内 |
| xlTickMarkNone    | -4142 | 无标志 |
| xlTickMarkOutside | 3     | 在轴外 |
