# XlToolbarProtection 枚举

指定工具栏的哪些属性受到限制。可用 Or 组合选项。

| 名称                    | 值    | 说明                       |
|-------------------------|-------|----------------------------|
| xlNoButtonChanges       | 1     | 不允许按钮更改。           |
| xlNoChanges             | 4     | 无任何类型的更改。         |
| xlNoDockingChanges      | 3     | 无对工具栏固定位置的更改。 |
| xlNoShapeChanges        | 2     | 无对工具栏形状的更改。     |
| xlToolbarProtectionNone | -4143 | 允许任何更改。             |
