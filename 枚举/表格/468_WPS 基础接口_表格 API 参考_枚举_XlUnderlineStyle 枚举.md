# XlUnderlineStyle 枚举

指定应用于字体的下划线类型。

| 名称                             | 值    | 说明                       |
|----------------------------------|-------|----------------------------|
| xlUnderlineStyleDouble           | -4119 | 粗双下划线。               |
| xlUnderlineStyleDoubleAccounting | 5     | 靠近在一起的两条细下划线。 |
| xlUnderlineStyleNone             | -4142 | 无下划线。                 |
| xlUnderlineStyleSingle           | 2     | 单下划线。                 |
| xlUnderlineStyleSingleAccounting | 4     | 不支持。                   |
