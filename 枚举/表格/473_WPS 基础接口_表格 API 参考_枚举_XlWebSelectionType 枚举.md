# XlWebSelectionType 枚举

指定是将整个网页、网页上的所有表还是特定表导入查询表。

| 名称              | 值  | 说明   |
|-------------------|-----|--------|
| xlAllTables       | 2   | 所有表 |
| xlEntirePage      | 1   | 整页   |
| xlSpecifiedTables | 3   | 指定表 |
