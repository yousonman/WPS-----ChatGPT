# XlXLMMacroType 枚举

指定在 ET 版本 4 宏工作表中，名称引用哪种宏，或名称是否引用宏。

| 名称       | 值  | 说明         |
|------------|-----|--------------|
| xlCommand  | 2   | 自定义命令。 |
| xlFunction | 1   | 自定义函数。 |
| xlNotXLM   | 3   | 非宏。       |
