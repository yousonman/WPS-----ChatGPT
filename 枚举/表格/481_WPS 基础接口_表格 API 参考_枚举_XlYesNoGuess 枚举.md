# XlYesNoGuess 枚举

指定第一行是否包含标题。不能在对数据透视表进行排序时使用。

| 名称    | 值  | 说明                                    |
|---------|-----|-----------------------------------------|
| xlGuess | 0   | ET 确定是否有标题，如果有，是否是一个。 |
| xlNo    | 2   | 默认值。应对整个区域进行排序。          |
| xlYes   | 1   | 不应对整个区域进行排序。                |
