# BackstageGroupStyle 枚举

指定用于定义 **“文件”** 选项卡上的组样式的常量。

| 名称                       | 值  | 说明       |
|----------------------------|-----|------------|
| BackstageGroupStyleError   | 2   | 错误样式。 |
| BackstageGroupStyleNormal  | 0   | 正常样式。 |
| BackstageGroupStyleWarning | 1   | 警告样式。 |
