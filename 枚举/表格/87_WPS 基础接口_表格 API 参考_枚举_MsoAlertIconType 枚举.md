# MsoAlertIconType 枚举

指定在警告中显示哪个图标（如果有）。

| 名称                 | 值  | 说明                     |
|----------------------|-----|--------------------------|
| msoAlertIconCritical | 1   | 显示 “重要” 图标。       |
| msoAlertIconInfo     | 4   | 显示 “信息” 图标。       |
| msoAlertIconNoIcon   | 0   | 在警告消息中不显示图标。 |
| msoAlertIconQuery    | 2   | 显示 “查询” 图标。       |
| msoAlertIconWarning  | 3   | 显示 “警告” 图标。       |
