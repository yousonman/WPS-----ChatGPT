# MsoArrowheadLength 枚举

指定线条末端的箭头长度。

| 名称                     | 值  | 说明                                         |
|--------------------------|-----|----------------------------------------------|
| msoArrowheadLengthMedium | 2   | 中等。                                       |
| msoArrowheadLengthMixed  | -2  | 只返回值，表示指定形状范围中其他状态的组合。 |
| msoArrowheadLong         | 3   | 长。                                         |
| msoArrowheadShort        | 1   | 短。                                         |
