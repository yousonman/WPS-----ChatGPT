# MsoArrowheadWidth 枚举

指定线条末端的箭头宽度。

| 名称                    | 值  | 说明                           |
|-------------------------|-----|--------------------------------|
| msoArrowheadNarrow      | 1   | 窄。                           |
| msoArrowheadWide        | 3   | 宽。                           |
| msoArrowheadWidthMedium | 2   | 中等。                         |
| msoArrowheadWidthMixed  | -2  | 只返回值，表示其他状态的组合。 |
