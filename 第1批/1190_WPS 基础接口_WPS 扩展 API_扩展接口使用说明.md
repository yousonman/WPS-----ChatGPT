# 扩展接口使用说明

**基本介绍**

WPS的扩展接口是对WPS现有接口的进一步扩充，大大的丰富了WPS的接口，以便满足二次开发各种解决方案的需要。WPS所有的扩展接口都满足唯一性、对称性、自反性、传递性，也就是说通过扩展接口可以查询到对应的原始接口，同样的通过原始接口也可以查到对应的扩展接口。
