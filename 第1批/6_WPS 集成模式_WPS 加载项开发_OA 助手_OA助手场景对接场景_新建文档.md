# 新建文档

创建空白文档，可以通过参数指定名称，格式转换保存，设置用户名等

``` JavaScript
_WpsStartUp([{
    "NewDoc": {}
}])
```

| 参数名 | 数据类型 | 描述                                     | 必选 |
|--------|----------|------------------------------------------|------|
| *NewDoc* | String   | 方法名，对应于OA助手dispatcher支持的方法 | 是   |
